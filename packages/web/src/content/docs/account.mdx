MongolGPT Web, Desktop болон CLI нь нэг MongolGPT бүртгэл, ажлын талбар, хэрэглээний эрхийг хуваалцах архитектуртай. Удирдлагатай загвар ашиглахад бүртгэлээр нэвтэрнэ. Өөрийн API түлхүүр болон локал загварын эрх нь үүнээс тусдаа байна.

:::caution[Одоогийн төлөв]
Нэгдсэн нэвтрэлт, Free Auto, quota, usage болон QPay/Bonum төлбөрийн код хөгжүүлэлтийн орчинд туршигдаж байна. Production үйлчилгээ, бодит төлбөр болон эцсийн үнэ албан ёсны release-д идэвхжтэл бодит төлбөрийн мэдээлэл бүү оруул. `*.dev.mgpt.mn` хаяг нь туршилтын орчин бөгөөд production үйлчилгээ биш.
:::

## Нэвтрэх

- **Web:** MongolGPT-ийн нэвтрэх дэлгэцээс Google эсвэл GitHub OAuth ашиглана.
- **Desktop:** системийн хөтөч нээгдэж, зөвшөөрсний дараа MongolGPT апп руу буцна.
- **CLI:** хөтөч эсвэл төхөөрөмжийн кодоор нэвтэрнэ.

```bash
mongolgpt console login
```

Нэвтэрсэн байгууллага, ажлын талбаруудаа шалгах:

```bash
mongolgpt console orgs
mongolgpt console switch
mongolgpt console open
```

CLI болон Desktop нь токеныг эх кодын тохиргоонд бичихгүй. Үйлдлийн системийн хамгаалалттай нууц мэдээллийн санд хадгалж, хугацаа дуусахад баталгаажсан auth service-ээр шинэчилнэ.

## MongolGPT Free Auto

MongolGPT Free Auto нь бодлогоор зөвшөөрсөн боломжтой загварыг автоматаар сонгоно. Энэ горим:

- зөвхөн нэвтэрсэн MongolGPT бүртгэлд ажиллана;
- хүсэлт, токен, зардал болон зэрэгцээ ажиллагааны хязгаарыг мөрдөнө;
- үйлчилгээ үзүүлэгчийн эрүүл мэнд, rate limit болон зардлын хамгаалалтаар чиглэлээ өөрчилж болно;
- туршилтын эсвэл хувь хүний үнэгүй API эрхийг нийтийн resale backend болгон ашиглахгүй;
- үүрд хязгааргүй, тогтмол нэг загвартай байна гэсэн амлалт биш.

Хязгаарт хүрвэл клиент дахин ашиглах хугацаа болон багцын хуудсыг Монгол хэлээр харуулна.

## Basic, Pro, Max

Basic, Pro, Max багцын quota, model access, request limit болон зардлын hard limit нь серверийн тохиргоогоор удирдагдана. Үнэ, татвар, буцаалт болон хүчинтэй нөхцөлийг код доторх хуучрах хүснэгтээр биш, нээлтийн үед бүртгэлийн самбар болон албан ёсны үйлчилгээний нөхцөлөөр харуулна.

Төлбөрийн урсгал QPay болон Bonum-д зориулагдсан. Merchant эрх, webhook болон reconciliation production-д идэвхжээгүй үед төлбөрийн товч ажиллаж байгаа мэт харагдах ёсгүй.

## Өөрийн үйлчилгээ үзүүлэгч

MongolGPT бүртгэлтэй байсан ч өөрийн загварыг сонгож болно:

- OpenRouter, OpenAI, Anthropic, Google болон бусад provider-ийн өөрийн API түлхүүр;
- өөрийн API түлхүүртэй NVIDIA NIM;
- OpenAI-тэй нийцтэй хувийн endpoint;
- Desktop эсвэл CLI дээр Ollama, LM Studio зэрэг локал загвар.

```txt
/connect
/models
```

BYOK түлхүүрийг MongolGPT-ийн source control, log эсвэл analytics-д бичихгүй. Локал загварын endpoint-ийг Web-ээс ашиглахдаа хамгаалалттай Desktop/local bridge шаардлагатай; нийтийн Web апп хэрэглэгчийн `localhost` руу шууд хандахгүй.

## Production-д бэлэн эсэхийг таних

Удирдлагатай үйлчилгээ production-д нээгдсэн гэж дараах бүх нөхцөл зэрэг биелсэн үед үзнэ:

1. Албан ёсны release notes-д нээлтийн төлөв, үнэ, quota болон нөхцөл нийтлэгдсэн.
2. `mgpt.mn`, `app.mgpt.mn` болон auth/API хаягууд хүчинтэй TLS-тэй, status check нь зөв JSON schema буцааж байгаа.
3. Desktop болон CLI-ийн production channel OAuth нэвтрэлт ажиллаж байгаа.
4. QPay/Bonum merchant production эрх идэвхжиж, туршилтын төлбөрийн reconciliation баталгаажсан.

Эдгээрээс өмнө Free Auto болон төлбөртэй багцыг зөвхөн хөгжүүлэлт, туршилтын орчинд ашиглана.
