MongolGPT-ийн ecosystem хуудсыг одоогоор зөвхөн баталгаажсан integration төрлүүдээр харуулж байна. Албан ёсны marketplace/package registry бэлэн болоогүй тул энд гуравдагч талын rebrand хийсэн package жагсаалт тавихгүй.

## Баталгаатай integration төрлүүд

| Төрөл | Одоогийн төлөв |
| --- | --- |
| MCP server | `mongolgpt compat import plan/apply` command болон desktop integration wizard-аар command, local path, URL, config-оос танина |
| Skill | Claude/Codex/Goose маягийн skill folder/config-ийг таних суурь importer нэмэгдсэн |
| Plugin | Local plugin file болон private npm package-ийг config дээр зааж ажиллуулах боломжтой |
| Provider | MongolGPT account login, API key provider, local/provider config wiring хадгалагдсан |

## Import workflow

```bash
mongolgpt compat import plan "npx -y @modelcontextprotocol/server-filesystem C:\\Users\\me"
mongolgpt compat import apply "npx -y @modelcontextprotocol/server-filesystem C:\\Users\\me"
```

Desktop дээр `Тохиргоо -> Интеграц` хэсгээс command, path, URL эсвэл config оруулаад шалгах болон нэмэх боломжтой.

## Нийтлэхээс өмнөх дүрэм

- Нийтлэгдээгүй npm/Homebrew/AUR/Chocolatey/Scoop package-ийг official install гэж харуулахгүй.
- Гуравдагч талын plugin-ийг MongolGPT нэрээр rebrand хийж, бодитоор байхгүй URL/package руу заахгүй.
- Шинэ integration нэмэхдээ эхлээд importer/adapter-аар шалгаж, дараа нь албан ёсны registry/marketplace дээр нийтэлсний дараа docs-д package install command нэмнэ.
