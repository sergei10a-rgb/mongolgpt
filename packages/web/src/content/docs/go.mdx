## Одоогийн төлөв

**MongolGPT Go одоогоор дэмжигдэхгүй.** Go provider catalog-д шаардлагатай MongolGPT-ийн өөрийн production backend, account, authentication, billing, subscription болон quota service нийтэд ашиглахад бэлэн болоогүй.

:::caution
Go нэртэй танихгүй домэйнд account үүсгэх, нэвтрэх мэдээлэл, API key эсвэл төлбөрийн мэдээлэл бүү оруул. MongolGPT одоогоор Go-ийн нийтийн URL, үнэ, subscription, credit эсвэл quota амлахгүй.
:::

CLI/TUI, local web болон desktop нь Go-оос хамаарахгүй. Нээлттэй model ашиглах бол [provider-ийн зааврын](../providers) дагуу Ollama зэрэг локал provider эсвэл өөрийн эрхтэй OpenRouter болон бусад BYOK provider-ийг холбоно.

```txt
/connect
/models
```

## Ирээдүйн production тохиргоо

Төслийн өөрийн backend бэлэн болж, release notes-д Go дэмжлэг албан ёсоор идэвхжсэн үед repository-ийн `.env.example` доторх `MONGOLGPT_GO_URL` болон бусад өөрийн URL-уудыг deployment-д тохируулна. Одоогоор hosted боломжууд идэвхгүй байна:

```dotenv
MONGOLGPT_ENABLE_HOSTED_SERVICES=false
```

`.env.example` доторх `https://your-domain.example` нь ажилладаг service биш, илэрхий placeholder юм. MongolGPT Zen, MongolGPT Black, account, billing болон public share мөн backend бэлэн болтол дэмжигдэхгүй.

Одоогийн дэмжигдсэн боломжуудыг [нээлтийн дэмжлэгийн хүрээнээс](../#нээлтийн-дэмжлэгийн-хүрээ) харна уу.
