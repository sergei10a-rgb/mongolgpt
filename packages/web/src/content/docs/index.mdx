[**MongolGPT**](/) бол терминал, ширээний програм, IDE, вэб интерфэйсээр ашиглах боломжтой AI кодын агент юм. Эх код, албан ёсны хувилбар, татах файлуудыг зөвхөн [GitHub Releases](https://github.com/sergei10a-rgb/mongolgpt/releases) болон төслийн [GitHub сангаас](https://github.com/sergei10a-rgb/mongolgpt) авна.

## Нээлтийн дэмжлэгийн хүрээ

Одоогийн release нь өөрийн компьютерт ажиллах дараах боломжуудад төвлөрнө.

| Төлөв | Боломж |
| --- | --- |
| Дэмжигдэнэ | [CLI](./cli) болон [TUI](./tui) |
| Дэмжигдэнэ | [`mongolgpt web`](./web)-ээр ажиллах локал web UI |
| Дэмжигдэнэ | GitHub Releases-д asset нь нийтлэгдсэн [Windows desktop](#ширээний-програм-суулгах) |
| Дэмжигдэнэ | Өөрийн API key эсвэл OAuth эрх ашиглах [BYOK provider](./providers) |
| Дэмжигдэнэ | [MCP](./mcp-servers) болон [LSP](./lsp) integration |
| Дэмжигдэхгүй | MongolGPT Zen, MongolGPT Go, MongolGPT Black |
| Дэмжигдэхгүй | MongolGPT account, billing, subscription, нийтийн share link |

:::caution[Hosted боломжууд идэвхгүй]
Zen, Go, Black, account, billing болон public share-д шаардлагатай MongolGPT-ийн өөрийн backend одоогоор бэлэн биш. Эдгээрт зориулсан танихгүй домэйнд нэвтрэх мэдээлэл, API key, төлбөрийн мэдээлэл бүү оруул. CLI/TUI, локал web, desktop болон BYOK provider ашиглахад домэйн эсвэл hosting хэрэггүй.
:::

## Аль хувилбарыг сонгох вэ?

- **Windows-ийн ширээний програм** хэрэгтэй бол Releases хуудасны `mongolgpt-desktop-win-x64.exe` файлыг сонгоно.
- **Терминалын CLI** хэрэгтэй бол үйлдлийн систем, процессортоо тохирох `mongolgpt-windows-*`, `mongolgpt-darwin-*`, эсвэл `mongolgpt-linux-*` архивыг сонгоно.
- **Хөгжүүлэлт хийх** эсвэл бэлэн файл байхгүй платформ дээр ажиллуулах бол эх кодоос ажиллуулна.

:::caution
Татах файлын нэр, хувилбар бүрд өөр байж болно. Release-ийн тайлбар болон файлын жагсаалтыг шалгаж, танихгүй домэйн эсвэл гуравдагч талын толин сайтаас суулгагч бүү тат.
:::

## Ширээний програм суулгах

1. [MongolGPT Releases](https://github.com/sergei10a-rgb/mongolgpt/releases) хуудсыг нээнэ.
2. Хамгийн сүүлийн тогтвортой release-ийг сонгоно.
3. `mongolgpt-desktop-win-x64.exe` файл тухайн release-д байвал татаж ажиллуулна.
4. Суулгасны дараа MongolGPT-ийг нээгээд төслийн хавтсаа сонгоно.
5. **Connect provider** хэсгээс өөрийн ашиглах үйлчилгээ үзүүлэгчийг холбоно.

Хэрэв Windows SmartScreen анхааруулга үзүүлбэл release-ийн URL нь `github.com/sergei10a-rgb/mongolgpt/releases` мөн эсэхийг дахин шалга. Release-ийн тайлбарт гарын үсэггүй build гэж тэмдэглэсэн бол байгууллагынхаа аюулгүй байдлын бодлогыг дагана.

Ширээний програм нээгдэхгүй, хоосон цонх харуулах, сервертэй холбогдохгүй байвал [алдаа оношлох зааврын ширээний програмын хэсгийг](./troubleshooting#ширээний-програм) үзнэ үү.

## CLI суулгах

1. [MongolGPT Releases](https://github.com/sergei10a-rgb/mongolgpt/releases) хуудсаас платформдоо тохирох архивыг татна.
2. Архивыг задлахад `mongolgpt` эсвэл Windows дээр `mongolgpt.exe` файл гарна.
3. Файлыг `PATH`-д бүртгэлтэй хавтас руу байрлуулах эсвэл тухайн хавтсаас шууд ажиллуулна.
4. Суулгалтыг шалгана.

```bash
mongolgpt --version
```

macOS болон Linux дээр ажиллуулах эрх шаардлагатай бол:

```bash
chmod +x mongolgpt
./mongolgpt --version
```

Windows PowerShell дээр:

```powershell
.\mongolgpt.exe --version
```

Package manager-ийн команд release-ийн тайлбарт албан ёсоор дурдагдсан үед л ашиглаарай. Баталгаатай үндсэн татах суваг нь GitHub Releases юм.

## Эх кодоос ажиллуулах

Энэ арга нь Bun болон Git суулгасан хөгжүүлэлтийн орчинд зориулагдсан.

```bash
git clone https://github.com/sergei10a-rgb/mongolgpt
cd mongolgpt
bun install
bun run dev
```

Ширээний програмыг хөгжүүлэлтийн горимоор ажиллуулах:

```bash
bun run dev:desktop
```

Windows суулгагч үүсгэх:

```powershell
$env:MONGOLGPT_CHANNEL="prod"
bun --cwd packages/desktop run package:win
```

Үүссэн файл `packages/desktop/dist/` хавтаст байрлана.

## Анхны ажиллагаа

CLI-г төслийнхөө хавтсаас эхлүүлнэ.

```bash
cd path/to/project
mongolgpt
```

Дараа нь:

1. `/connect` командаар үйлчилгээ үзүүлэгчээ сонгож API түлхүүр эсвэл тухайн үйлчилгээний нэвтрэх аргыг тохируулна.
2. `/models` командаар ашиглах боломжтой загваруудаа шалгана.
3. Загвараа сонгоод хийх ажлаа монгол эсвэл өөрийн хүссэн хэлээр тодорхой бичнэ.
4. Файл өөрчлөх, команд ажиллуулах зөвшөөрлийн хүсэлтийг уншаад зөвшөөрөх эсэхээ шийднэ.

```txt
/connect
/models
Энэ төслийн тест яагаад унаж байгааг шалгаад зас.
```

MongolGPT account одоогоор дэмжигдэхгүй бөгөөд үндсэн ажиллагаанд хэрэггүй. OpenAI, Anthropic, Google, OpenRouter болон бусад дэмжигдсэн provider-ийг өөрийн API key эсвэл provider-ийн OAuth эрхээр холбоно. Дэлгэрэнгүйг [үйлчилгээ үзүүлэгчдийн заавраас](./providers) үзнэ үү.

## Домэйн болон байршуулалт хэрэгтэй юу?

Үгүй. Одоогийн launch profile бүхэлдээ standalone/local тул MongolGPT ашиглахын тулд production domain, DNS эсвэл hosting тохируулах шаардлагагүй.

Ирээдүйд төслийн өөрийн production backend бэлэн болж hosted боломжуудыг байршуулбал repository-ийн `.env.example`-ийг суурь болгон `MONGOLGPT_PUBLIC_URL`, `MONGOLGPT_AUTH_URL`, `MONGOLGPT_API_URL`, `MONGOLGPT_SHARE_URL`, `MONGOLGPT_GO_URL` зэрэг утгыг зөвхөн өөрийн эзэмшлийн URL-аар солино. Тэр хүртэл `MONGOLGPT_ENABLE_HOSTED_SERVICES=false` хэвээр байна. `.env.example` доторх `https://your-domain.example` нь ажилладаг service биш, илэрхий placeholder юм.
