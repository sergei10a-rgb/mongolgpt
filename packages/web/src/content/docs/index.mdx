[**MongolGPT**](/) бол Монгол хэрэглэгчдэд зориулсан AI coding agent платформ. Терминал, desktop app, IDE өргөтгөл, provider, skill/plugin, MCP холболтуудыг нэг дор ажиллуулах зорилготой.

## Одоогийн төлөв

MongolGPT-ийн Windows desktop туршилтын build GitHub Release дээр нийтлэгдсэн.

[MongolGPT Desktop татах](https://github.com/sergei10a-rgb/mongolgpt/releases/latest)

`npm install -g mongolgpt`, Homebrew, Chocolatey, Scoop package-уудыг албан ёсоор нийтлэх ажил бэлтгэгдэж байна. Эдгээр package registry дээр гараагүй үед docs дээр шууд суулгана гэж бичих нь буруу тул энэ хуудсанд зөвхөн бодитой ажиллах замуудыг үлдээв.

## Desktop суулгах

1. [Releases](https://github.com/sergei10a-rgb/mongolgpt/releases/latest) хуудас руу орно.
2. `mongolgpt-desktop-win-x64.exe` файлыг татна.
3. Суулгагчийг ажиллуулна.

:::note
Одоогийн build code signing хийгдээгүй. Windows SmartScreen анхааруулга харуулж магадгүй.
:::

## Source-оос ажиллуулах

Repository-г clone хийсэн бол дараах байдлаар локал орчинд ажиллуулна.

```bash
bun install
bun run dev
```

Desktop development mode:

```bash
bun run dev:desktop
```

Windows desktop installer build хийх:

```powershell
$env:MONGOLGPT_CHANNEL="prod"
bun --cwd packages/desktop run package:win
```

Build дууссаны дараа installer файл энд гарна.

```txt
packages/desktop/dist/
```

## Терминал хувилбар

Terminal CLI нь `mongolgpt` command нэрээр ажиллах ёстой. Repository дотор CLI болон publish script-үүд энэ нэр рүү шилжсэн.

Registry package нийтлэгдсэний дараа суулгах үндсэн command:

```bash
npm install -g mongolgpt
```

Package нийтлэгдээгүй үед GitHub Release эсвэл source build ашиглана.

## Нэвтрэх ба provider тохируулах

MongolGPT account-аар нэвтрэх урсгал нь `https://mongolgpt.duckdns.org/auth` endpoint руу чиглэнэ.

```txt
/connect
```

`/connect` командаар provider сонгоод API key эсвэл MongolGPT account холболтоо тохируулна.

## Skill, plugin, MCP

MongolGPT-ийн зорилго нь Claude, Codex, Goose, MCP server, skill/plugin зэрэг өөр өөр AI agent format-уудыг хэрэглэгчээр гараар тааруулахгүйгээр таньж, шаардлагатай үед adapter/wrapper үүсгэн холбох юм.

Эхний суурь нь repo дотор байна:

- provider integration
- plugin API суурь
- MCP холболтын суурь
- desktop shell
- terminal agent
- MongolGPT account login wiring

Дараагийн шатанд format танигч, adapter generator, marketplace/import workflow нэмэгдэнэ.
