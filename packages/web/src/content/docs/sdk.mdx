mongolgpt JS/TS SDK нь сервертэй харьцах төрлийн аюулгүй клиентээр хангадаг.
Интеграцчлалыг бий болгох, mongolgpt-г програмын дагуу хянахын тулд үүнийг ашиглана уу.

[Илүү ихийг мэдэж аваарай](/docs/server) сервер хэрхэн ажилладаг талаар. Жишээлбэл, нийгэмлэгийн бүтээсэн [projects](/docs/ecosystem#projects)-г үзээрэй.

---

## SDK ашиглах

`@mongolgpt/sdk` package npm дээр хараахан нийтлэгдээгүй. Одоогоор SDK-г энэ monorepo workspace дотроос ашиглана.

```bash
git clone https://github.com/sergei10a-rgb/mongolgpt
cd mongolgpt
bun install
```

Repository дотор `@mongolgpt/sdk` import нь workspace package-ээр resolved болно. Package нийтлэгдсэний дараа npm install command-ийг энэ хуудсанд нэмнэ.

## Үйлчлүүлэгч үүсгэх

mongolgpt жишээ үүсгэх:

```javascript
import { createMongolGPT } from "@mongolgpt/sdk"

const { client } = await createMongolGPT()
```

Энэ нь сервер болон үйлчлүүлэгчийг хоёуланг нь эхлүүлнэ

#### Сонголтууд

| Сонголт | Төрөл | Тодорхойлолт | Өгөгдмөл |
| ---------- | ------------- | ------------------------------ | ----------- |
| `hostname` | `string` | Серверийн хостын нэр | `127.0.0.1` |
| `port` |
umber` | Сервер порт | `4096` |
| `signal` | `AbortSignal` | Цуцлахыг зогсоох дохио | `undefined` |
| `timeout` |
umber` | Серверийг эхлүүлэхэд ms-ийн хугацаа | `5000` |
| `config` | `Config` | Тохиргооны объект | `{}` |

## Тохиргоо

Та зан төлөвийг өөрчлөхийн тулд тохиргооны объектыг дамжуулж болно. Энэ жишээ нь таны `mongolgpt.json`-г авсан хэвээр байгаа ч та тохиргоог дарж бичих эсвэл нэмэх боломжтой:

```javascript
import { createMongolGPT } from "@mongolgpt/sdk"

const mongolgpt = await createMongolGPT({
  hostname: "127.0.0.1",
  port: 4096,
  config: {
    model: "anthropic/claude-3-5-sonnet-20241022",
  },
})

console.log(`Server running at ${mongolgpt.server.url}`)

mongolgpt.server.close()
```

## Зөвхөн үйлчлүүлэгч

Хэрэв танд mongolgpt-ийн ажиллаж байгаа жишээ байгаа бол та түүнтэй холбогдохын тулд үйлчлүүлэгчийн жишээ үүсгэж болно:

```javascript
import { createMongolGPTClient } from "@mongolgpt/sdk"

const client = createMongolGPTClient({
  baseUrl: "http://localhost:4096",
})
```

#### Сонголтууд

| Сонголт | Төрөл | Тодорхойлолт | Өгөгдмөл |
| --------------- | ---------- | -------------------------------- | ----------------------- |
| `baseUrl` | `string` | URL серверийн | `http://localhost:4096` |
| `fetch` | `function` | Захиалгат дуудлагын хэрэгжилт | `globalThis.fetch` |
| `parseAs` | `string` | Хариултыг задлан шинжлэх арга | `auto` |
| `responseStyle` | `string` | Буцах загвар: `data` эсвэл `fields` | `fields` |
| `throwOnError` | `boolean` | Буцаахын оронд алдаа шидэх | `false` |

## Бүтэцлэгдсэн гаралт

Та JSON схем бүхий `format`-г зааж өгснөөр загвараас бүтэцлэгдсэн JSON гаралтыг хүсч болно. Загвар нь таны схемд тохирсон JSON-г буцаахын тулд `StructuredOutput` хэрэгслийг ашиглана.

### Үндсэн хэрэглээ

```typescript
const result = await client.session.prompt({
  path: { id: sessionId },
  body: {
    parts: [{ type: "text", text: "Research Anthropic and provide company info" }],
    format: {
      type: "json_schema",
      schema: {
        type: "object",
        properties: {
          company: { type: "string", description: "Company name" },
          founded: { type: "number", description: "Year founded" },
          products: {
            type: "array",
            items: { type: "string" },
            description: "Main products",
          },
        },
        required: ["company", "founded"],
      },
    },
  },
})

// Access the structured output
console.log(result.data.info.structured_output)
// { company: "Anthropic", founded: 2021, products: ["Claude", "Claude API"] }
```

### Гаралтын форматын төрлүүд

| Төрөл | Тодорхойлолт |
| ------------- | ------------------------------------------------------ |
| `text` | Өгөгдмөл. Стандарт текстийн хариу (бүтэцлэгдсэн гаралт байхгүй) |
| `json_schema` | Өгөгдсөн схемтэй тохирч баталгаажуулсан JSON буцаана |

### JSON Схемийн формат

`type: 'json_schema'`-г ашиглахдаа:

| Талбай | Төрөл | Тодорхойлолт |
| ------------ | --------------- | ---------------------------------------------------------- |
| `type` | `'json_schema'` | Шаардлагатай. JSON схемийн горимыг заана |
| `schema` | `object` | Шаардлагатай. JSON Гаралтын бүтцийг тодорхойлох схемийн объект |
| `retryCount` |
umber` | Нэмэлт. Баталгаажуулалтын дахин оролдлогын тоо (өгөгдмөл: 2) |

### Алдаа засах

Хэрэв загвар бүх дахин оролдлого хийсний дараа хүчинтэй бүтэцтэй гаралтыг гаргаж чадахгүй бол хариулт нь `StructuredOutputError`-г агуулна:

```typescript
if (result.data.info.error?.name === "StructuredOutputError") {
  console.error("Failed to produce structured output:", result.data.info.error.message)
  console.error("Attempts:", result.data.info.error.retries)
}
```

### Шилдэг туршлага

1. Загварт ямар өгөгдөл гаргаж авахыг ойлгоход туслахын тулд схемийн шинж чанартаа **тодорхой тайлбаруудыг оруулна уу**
2. ** `required`**-г ашиглан аль талбарууд байх ёстойг зааж өгнө үү
3. **Схемүүдийг анхаарлаа төвлөрүүлэх** - нийлмэл үүрлэсэн схемүүд нь загварт зөв бөглөхөд хэцүү байж болно.
4. **Тохирохыг тохируулна уу `retryCount`** - нарийн төвөгтэй схемийн хувьд нэмэгдүүлэх, энгийн схемийн хувьд бууруулна.

### Сеанс

| арга | Тодорхойлолт | Тэмдэглэл |
| ---------------------------------------------------------- | ---------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `session.list()` | Жагсаалт сесс | <a href={typesUrl}><code>Session[]</code></a>-г буцаана |

| `session.children({ path })` | Хүүхдийн сессүүдийг жагсаах | <a href={typesUrl}><code>Session[]</code></a>-г буцаана |
| `session.create({ body })` | Сеанс үүсгэх | <a href={typesUrl}><code>Session</code></a>-г буцаана |
| `session.delete({ path })` | Сеанс устгах | `boolean` \| буцаана |
| `session.update({ path, body })` | Сеансын шинж чанарыг шинэчлэх | <a href={typesUrl}><code>Session</code></a>-г буцаана |
| `session.init({ path, body })` | Програмд ​​дүн шинжилгээ хийж, `AGENTS.md` | үүсгэнэ үү `boolean` \| буцаана |

| `session.message({ path })` | Мессежийн дэлгэрэнгүй мэдээллийг авах | `{ info: `<a href={typesUrl}><code>Мессеж</code></a>`, parts: `<a href={typesUrl}><code>Хэсэг[]</code></a>`}` буцаана |

| `session.unrevert({ path })` | Буцаасан мессежүүдийг сэргээх | <a href={typesUrl}><code>Session</code></a>-г буцаана |
| `postSessionByIdPermissionsByPermissionId({ path, body })` | Зөвшөөрлийн хүсэлтэд хариу өгөх | `boolean` \| буцаана |

---

#### Жишээ

```javascript
// Create and manage sessions
const session = await client.session.create({
  body: { title: "My session" },
})

const sessions = await client.session.list()

// Send a prompt message
const result = await client.session.prompt({
  path: { id: session.id },
  body: {
    model: { providerID: "anthropic", modelID: "claude-3-5-sonnet-20241022" },
    parts: [{ type: "text", text: "Hello!" }],
  },
})

// Inject context without triggering AI response (useful for plugins)
await client.session.prompt({
  path: { id: session.id },
  body: {
    noReply: true,
    parts: [{ type: "text", text: "You are a helpful assistant." }],
  },
})
```

### Файлууд

| арга | Тодорхойлолт | Хариулт |
| ------------------------- | ---------------------------------- | ------------------------------------------------------------------------------------------- |
| `find.text({ query })` | Файлаас текст хайх | `path`, `lines`, `line_number`, `absolute_offset`, `submatches`-тай таарах объектуудын массив |
| `find.files({ query })` | Файл, санг нэрээр нь хайж олох | `string[]` (замууд) |
| `find.symbols({ query })` | Ажлын талбарын тэмдэгтүүдийг олох | <a href={typesUrl}><code>Тэмдэгт[]</code></a> |
| `file.read({ query })` | Файл унших | `{ type: "raw" \| "patch", content: string }` |
| `file.status({ query? })` | Хянагдсан файлуудын статусыг авах | <a href={typesUrl}><code>Файл[]</code></a> |

`find.files` нь хэд хэдэн нэмэлт асуулгын талбаруудыг дэмждэг:

- `type`: `"file"` эсвэл `"directory"`
- `directory`: хайлтын төслийн үндэсийг хүчингүй болгох
- `limit`: хамгийн их үр дүн (1–200)

---

#### Жишээ

```javascript
// Search and read files
const textResults = await client.find.text({
  query: { pattern: "function.*mongolgpt" },
})

const files = await client.find.files({
  query: { query: "*.ts", type: "file" },
})

const directories = await client.find.files({
  query: { query: "packages", type: "directory", limit: 20 },
})

const content = await client.file.read({
  query: { path: "src/index.ts" },
})
```
