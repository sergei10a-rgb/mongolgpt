MongolGPT нь стандарт прокси орчны хувьсагч болон байгууллагын сүлжээний орчинд зориулсан тусгай гэрчилгээг дэмждэг.

---

## Прокси

MongolGPT стандарт прокси орчны хувьсагчдыг хүндэтгэдэг.

```bash
# HTTPS proxy (санал болгоно)
export HTTPS_PROXY=https://proxy.example.com:8080

# HTTPS боломжгүй үед HTTP proxy
export HTTP_PROXY=http://proxy.example.com:8080

# Локал серверийг proxy-оос алгасана (шаардлагатай)
export NO_PROXY=localhost,127.0.0.1
```

:::caution
TUI нь орон нутгийн HTTP сервертэй холбогддог. Чиглүүлэлтийн гогцооноос сэргийлэхийн тулд та энэ холболтын проксиг тойрч гарах ёстой.
:::

Та [CLI тугууд](../cli#run) ашиглан серверийн порт болон хостын нэрийг тохируулах боломжтой.

---

### Баталгаажуулах

Хэрэв таны прокси үндсэн баталгаажуулалт шаарддаг бол URL-д итгэмжлэлүүдийг оруулна уу.

```bash
export HTTPS_PROXY=http://username:password@proxy.example.com:8080
```

:::caution
Хатуу кодлох нууц үгээс зайлсхий. Хүрээлэн буй орчны хувьсагч эсвэл найдвартай итгэмжлэлийн хадгалах санг ашиглах.
:::

NTLM эсвэл Kerberos зэрэг дэвшилтэт нэвтрэлт танилт шаарддаг проксигийн хувьд таны баталгаажуулалтын аргыг дэмждэг LLM гарцыг ашиглахыг бодоорой.

---

## Тусгай гэрчилгээ

Хэрэв танай байгууллага HTTPS холболтод захиалгат CA ашигладаг бол тэдгээрт итгэхийн тулд MongolGPT-г тохируулна уу.

```bash
export NODE_EXTRA_CA_CERTS=/path/to/ca-cert.pem
```

Энэ нь прокси холболт болон шууд API хандалтад хоёуланд нь ажилладаг.
