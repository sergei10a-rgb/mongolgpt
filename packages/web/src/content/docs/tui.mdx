MongolGPT интерактив терминал интерфэйс буюу TUI LLM ашиглан төсөл дээрээ ажиллах боломжтой.

MongolGPT-г ажиллуулах нь одоогийн лавлахын TUI-г эхлүүлнэ.

```bash
mongolgpt
```

Эсвэл та үүнийг тодорхой ажлын лавлахын тулд эхлүүлж болно.

```bash
mongolgpt /path/to/project
```

TUI-д орсны дараа та үүнийг мессежээр сануулж болно.

```text
Кодын санг товч хураангуйл.
```

---

## Файлын лавлагаа

Та `@` ашиглан мессежийнхээ файлуудыг лавлаж болно. Энэ нь одоогийн ажиллаж байгаа лавлахаас тодорхой бус файл хайлт хийдэг.

:::tip
Та мөн `@` ашиглан өөрийн мессеж дэх файлуудыг лавлаж болно.
:::

```text "@packages/functions/src/api/index.ts"
`@packages/functions/src/api/index.ts` дотор auth-ийг хэрхэн боловсруулдаг вэ?
```

Файлын агуулгыг харилцан ярианд автоматаар нэмнэ.

Тохируулсан [references](../references) нь мөн `@` автомат бөглөх хэсэгт харагдана. Лавлагааны язгуурыг контекст болгон нэмэхийн тулд `@alias` гэж бичнэ үү, эсвэл тухайн лавлагааны доторх файлуудыг автоматаар бөглөхийн тулд `@alias/` гэж бичнэ үү.

```text "@docs/README.md"
Манай тохиргоог `@docs/README.md`-тэй харьцуул
```

## Bash командууд

Бүрхүүлийн командыг ажиллуулахын тулд `!` гэсэн мессежийг эхлүүлнэ үү.

```bash frame="none"
!ls -la
```

Хэрэгслийн үр дүнд командын гаралтыг харилцан ярианд нэмнэ.

## Командууд

MongolGPT TUI-г ашиглахдаа үйлдлүүдийг хурдан гүйцэтгэхийн тулд `/` дараа нь тушаалын нэрийг бичиж болно. Жишээ нь:

```bash frame="none"
/help
```

Ихэнх командууд нь `ctrl+x`-г өгөгдмөл удирдагч товчлуур болгон ашигладаг гарын товчлолтой байдаг. [Илүү ихийг мэдэж аваарай](../keybinds).

Энд байгаа бүх налуу зураастай командууд байна:

---

### холбогдоно уу

MongolGPT-д үйлчилгээ үзүүлэгч нэмнэ үү. Боломжтой үйлчилгээ үзүүлэгчдээс сонгох, тэдгээрийн API түлхүүрийг нэмэх боломжийг танд олгоно.

```bash frame="none"
/connect
```

---

### авсаархан

Одоогийн сессийг нягтруулна уу. _Алиас_: `/summarize`

```bash frame="none"
/compact
```

**Товчлуур:** `ctrl+x c`

---

### дэлгэрэнгүй

Хэрэгслийн гүйцэтгэлийн дэлгэрэнгүй мэдээллийг сэлгэх.

```bash frame="none"
/details
```

---

### редактор

Мессеж бичихийн тулд гадаад editor нээнэ. `EDITOR` орчны хувьсагчид заасан command-ийг ашиглана. [Editor тохиргоог](#редакторын-тохиргоо) харна уу.

```bash frame="none"
/editor
```

**Товчлуур:** `ctrl+x e`

---

### гарц

MongolGPT гарна. _Алиас_: `/quit`, `/q`

```bash frame="none"
/exit
```

**Товчлуур:** `ctrl+x q`

---

### экспорт

Одоогийн харилцан яриаг Markdown файл болгон экспортолж, `EDITOR` хувьсагчид заасан editor-т нээнэ. [Editor тохиргоог](#редакторын-тохиргоо) харна уу.

```bash frame="none"
/export
```

**Товчлуур:** `ctrl+x x`

---

### тусламж

Тусламжийн харилцах цонхыг харуул.

```bash frame="none"
/help
```

---

### эхлүүл

`AGENTS.md` үүсгэх эсвэл шинэчлэх зааварчилгаатай тохиргоо. [Илүү ихийг мэдэж аваарай](../rules).

```bash frame="none"
/init
```

---

### загвар

Боломжтой загваруудыг жагсаах.

```bash frame="none"
/models
```

**Товчлуур:** `ctrl+x m`

---

### шинэ

Шинэ сесс эхлүүлэх. _Алиас_: `/clear`

```bash frame="none"
/new
```

**Товчлуур:** `ctrl+x n`

---

### /redo

Өмнө нь болиулсан мессежийг дахин хийнэ үү. `/undo`-г ашигласны дараа л боломжтой.

:::tip
Аливаа файлын өөрчлөлт мөн сэргээгдэх болно.
:::

Файлын өөрчлөлтийг сэргээхэд Git ашигладаг тул төсөл Git repository байх шаардлагатай.

```bash frame="none"
/redo
```

**Товчлуур:** `ctrl+x r`

---

### /sessions

Жагсаж, сешн хооронд шилжих. _Алиас_: `/resume`, `/continue`

```bash frame="none"
/sessions
```

**Товчлуур:** `ctrl+x l`

---

### /share

Client команд байгаа боловч шаардлагатай public share backend одоогоор дэмжигдэхгүй. Сесс дамжуулахдаа `/export` ашиглана. [Share-ийн төлөв](../share)-ийг харна уу.

```bash frame="none"
/share
```

---

### /themes

Боломжтой сэдвүүдийг жагсаах.

```bash frame="none"
/themes
```

**Товчлуур:** `ctrl+x t`

---

### /thinking

Харилцан ярианы reasoning block-уудыг харуулах эсэхийг сэлгэнэ.

:::note
Энэ команд зөвхөн reasoning block **харагдах** эсэхийг өөрчилнө; model-ийн reasoning capability-г асааж, унтраахгүй. Model variant солих бол `ctrl+t` ашиглана.
:::

```bash frame="none"
/thinking
```

---

### /undo

Харилцан ярианы сүүлийн зурвасыг буцаах. Хэрэглэгчийн хамгийн сүүлийн мессеж, дараачийн бүх хариулт болон аливаа файлын өөрчлөлтийг устгана.

:::tip
Аливаа файлын өөрчлөлтийг мөн буцаах болно.
:::

Файлын өөрчлөлтийг буцаахад Git ашигладаг тул төсөл Git repository байх шаардлагатай.

```bash frame="none"
/undo
```

**Товчлуур:** `ctrl+x u`

---

### /unshare

Public share backend дэмжигдэхгүй тул launch profile-д энэ команд шаардлагагүй. Тохиргоонд `"share": "disabled"` ашиглана. [Share-ийн төлөв](../share)-ийг харна уу.

```bash frame="none"
/unshare
```

## Редакторын тохиргоо

`/editor` болон `/export` командууд нь `EDITOR` орчны хувьсагчд заасан засварлагчийг ашигладаг.

<Tabs>
  <TabItem label="Linux/macOS">
    ```bash
    # nano эсвэл vim ашиглах жишээ
    export EDITOR=nano
    export EDITOR=vim

    # VS Code, Cursor, VSCodium, Windsurf, Zed зэрэг GUI editor-т
    # --wait сонголтыг нэмнэ
    export EDITOR="code --wait"
    ```

    Байнгын болгохын тулд `~/.bashrc`, `~/.zshrc` зэрэг shell profile-д нэмнэ.

  </TabItem>

  <TabItem label="Windows (CMD)">
    ```bash
    set EDITOR=notepad

    # VS Code, Cursor, VSCodium, Windsurf, Zed зэрэг GUI editor-т
    # --wait сонголтыг нэмнэ
    set EDITOR=code --wait
    ```

    Байнгын болгохын тулд **System Properties** > **Environment Variables** хэсэгт `EDITOR` нэмнэ.

  </TabItem>

  <TabItem label="Windows (PowerShell)">
    ```powershell
    $env:EDITOR = "notepad"

    # VS Code, Cursor, VSCodium, Windsurf, Zed зэрэг GUI editor-т
    # --wait сонголтыг нэмнэ
    $env:EDITOR = "code --wait"
    ```

    Үүнийг байнгын болгохын тулд үүнийг PowerShell профайлдаа нэмнэ үү.

  </TabItem>
</Tabs>

Түгээмэл editor командууд:

- `code` - Visual Studio Code
- `cursor` - Cursor
- `windsurf` - Windsurf
- `nvim` - Neovim
- `vim` - Vim
- `nano` - Nano
- `notepad` - Windows Notepad
- `subl` - Sublime Text

:::note
VS Code зэрэг GUI editor-ийг `--wait` сонголттой эхлүүлэх шаардлагатай байж болно.
:::

Зарим редакторуудад блоклох горимд ажиллахын тулд командын мөрийн аргументууд хэрэгтэй. `--wait` туг нь засварлагчийн процессыг хаах хүртэл блоклодог.

## Тохируулах

Та `tui.json` (эсвэл `tui.jsonc`)-ээр дамжуулан TUI зан төлөвийг өөрчлөх боломжтой.

```json title="tui.json"
{
  "theme": "mongolgpt",
  "leader_timeout": 2000,
  "keybinds": {
    "leader": "ctrl+x",
    "command_list": "ctrl+p"
  },
  "scroll_speed": 3,
  "scroll_acceleration": {
    "enabled": false
  },
  "diff_style": "auto",
  "mouse": true,
  "attention": {
    "enabled": true,
    "notifications": true,
    "sound": true,
    "volume": 0.4,
    "sound_pack": "mongolgpt.default",
    "sounds": {
      "error": "./sounds/error.mp3"
    }
  }
}
```

Энэ нь сервер/ажиллах цагийн горимыг тохируулдаг `mongolgpt.json`-аас тусдаа юм.

`keybinds` нь анхдагч тохиргоотой нэгтгэгдсэн тул та зөвхөн өөрчлөхийг хүссэн товчлолуудаа тохируулах хэрэгтэй.

### Сонголтууд

- `theme` - Таны UI загварыг тохируулна. [Илүү ихийг мэдэж аваарай](../themes).
- `keybinds` - Гарын товчлолыг тохируулна. [Илүү ихийг мэдэж аваарай](../keybinds).
- `leader_timeout` - Удирдагч товчлуурын дараа MongolGPT хэр удаан хүлээхийг хянадаг. Өгөгдмөл нь `2000`.
- `scroll_acceleration.enabled` - Гөлгөр, байгалийн гүйлгэхийн тулд macOS загварын гүйлгэх хурдатгалыг идэвхжүүлнэ. Идэвхжүүлсэн үед гүйлгэх хурд нь хурдан гүйлгэх дохиогоор нэмэгдэж, удаан хөдөлгөөн хийхэд нарийвчлалтай хэвээр байна. **Энэ тохиргоо нь `scroll_speed`-ээс давуу эрхтэй бөгөөд идэвхжүүлсэн үед үүнийг хүчингүй болгодог.**
- `scroll_speed` - Гүйлгэх командуудыг ашиглах үед TUI хэр хурдан гүйлгэхийг хянадаг (хамгийн бага: `0.001`, аравтын утгыг дэмждэг). Өгөгдмөл нь `3`. **Тэмдэглэл: Хэрэв `scroll_acceleration.enabled`-г `true` гэж тохируулсан бол үүнийг хэрэгсэхгүй.**
- `diff_style` - Ялгаатай дүрслэлийг хянадаг. `"auto"` терминалын өргөнд дасан зохицдог, `"stacked"` нь үргэлж нэг баганын байршлыг харуулдаг.
- `mouse` - TUI (өгөгдмөл: `true`) дотор хулганаар авах ажиллагааг идэвхжүүлэх эсвэл идэвхгүй болгох. Идэвхгүй болсон үед терминалын үндсэн хулганы сонголт/гүйлгэх үйлдэл хадгалагдана.
- `attention` - TUI ширээний мэдэгдэл болон дууг тохируулна. Өгөгдмөлөөр идэвхгүй болгосон.

Захиалгат TUI тохиргооны замыг ачаалахын тулд `MONGOLGPT_TUI_CONFIG` ашиглана уу.

### Анхаарал

TUI нь асуулт, зөвшөөрөл, сессийн алдаа болон дууссан сешнүүдэд анхаарал хандуулах хүсэлт гаргаж болно. Үүнийг `attention.enabled` ашиглан идэвхжүүлнэ үү; суурилуулсан үйл явдлууд идэвхжсэн үед дууг тоглуулдаг ба дэд бус үйл явдлууд зөвхөн терминал бүдэгрэх үед ширээний мэдэгдлүүдийг шаарддаг.

- `enabled` - Анхаарал татах бүх мэдэгдэл, дууг идэвхжүүлнэ. Өгөгдмөл нь `false`.
- `notifications` - Анхаарлыг идэвхжүүлсэн үед terminal bridge-ээр desktop notification харуулахыг зөвшөөрнө. Өгөгдмөл нь `true`.
- `sound` - Анхаарал идэвхжсэн үед анхаарал хандуулахыг зөвшөөрөх. Өгөгдмөл нь `true`.
- `volume` - Анхдагч дууны хэмжээ нь `0`-аас `1` хүртэл. Өгөгдмөл нь `0.4`.
- `sound_pack` - Ашиглах дууны багц ID. Өгөгдмөл нь `mongolgpt.default`.
- `sounds` - ​​`default`, `question`, `permission`, `error`, `done`, эсвэл `subagent_done` дууны файлуудыг хүчингүй болгоно. Замууд нь үнэмлэхүй, `file://` URLs эсвэл `tui.json`-тэй харьцангуй байж болно.

## Тохируулга

Та тушаалын палитр (`ctrl+p`) ашиглан TUI харагдацын янз бүрийн талыг өөрчлөх боломжтой. Эдгээр тохиргоо нь дахин эхлүүлэх үед хэвээр байна.

---

#### Хэрэглэгчийн нэрийн дэлгэц

Таны хэрэглэгчийн нэр чатын зурваст гарч ирэх эсэхийг асаана уу. Үүнд:

- Тушаалын палитр: "хэрэглэгчийн нэр" эсвэл "хэрэглэгчийн нэрийг нуух" гэж хайх.
- Тохиргоо автоматаар хэвээр байх бөгөөд TUI сессүүдэд санах болно
