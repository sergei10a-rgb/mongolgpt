---
title: IDE өргөтгөл
description: VS Code, Cursor болон бусад дэмжигдсэн IDE-д MongolGPT ашиглах заавар.
---

MongolGPT нь VS Code болон түүнд суурилсан Cursor, Windsurf, VSCodium зэрэг IDE-ийн нэгдсэн терминалтай ажиллана. Эхлэхээсээ өмнө [CLI-г суулгаж](../), `mongolgpt --version` команд амжилттай ажиллаж байгааг шалгаарай.

---

## Ашиглах

- **Түргэн нээх**: macOS дээр `Cmd+Esc`, Windows/Linux дээр `Ctrl+Esc` дарж MongolGPT терминалыг нээнэ. Аль хэдийн ажиллаж байвал тухайн терминал руу шилжинэ.
- **Шинэ сесс**: macOS дээр `Cmd+Shift+Esc`, Windows/Linux дээр `Ctrl+Shift+Esc` дарж шинэ MongolGPT терминал эхлүүлнэ.
- **Контекст дамжуулах**: IDE-д сонгосон код эсвэл идэвхтэй файлын мэдээллийг MongolGPT-д дамжуулж болно.
- **Файл дурдах**: macOS дээр `Cmd+Option+K`, Windows/Linux дээр `Alt+Ctrl+K` дарж `@File#L37-42` хэлбэрийн файлын лавлагаа оруулна.

---

## Суулгах

1. VS Code, Cursor, Windsurf эсвэл VSCodium-ийг нээнэ.
2. Нэгдсэн терминалыг нээнэ.
3. `mongolgpt` командыг ажиллуулна. Дэмжигдсэн IDE-д өргөтгөл автоматаар суух боломжтой.

Автоматаар суухгүй бол IDE-ийн extension marketplace-оос **MongolGPT** гэж хайж, **Install** дарна.

TUI-ийн `/editor` болон `/export` командтай IDE editor ашиглах бол `EDITOR` хувьсагчид хүлээх сонголтыг хамт өгнө:

```bash
export EDITOR="code --wait"
```

Windows PowerShell дээр:

```powershell
$env:EDITOR = "code --wait"
```

Бусад editor-ийн жишээг [TUI-ийн editor тохиргоо](../tui/#редакторын-тохиргоо)-оос харна уу.

---

## Алдаа шийдэх

Өргөтгөл нээгдэхгүй эсвэл автоматаар суухгүй бол:

1. Нэгдсэн терминалд `mongolgpt --version` ажиллаж байгаа эсэхийг шалгана.
2. IDE-ийн CLI команд `PATH`-д байгааг шалгана: VS Code-д `code`, Cursor-д `cursor`, Windsurf-д `windsurf`, VSCodium-д `codium`.
3. IDE-ийн Command Palette-ийг macOS дээр `Cmd+Shift+P`, Windows/Linux дээр `Ctrl+Shift+P`-ээр нээгээд **Shell Command: Install 'code' command in PATH** эсвэл тухайн IDE-ийн ижил утгатай командыг ажиллуулна.
4. Байгууллагын policy extension суулгахыг хориглосон эсэх, мөн terminal profile өөр `PATH` ашиглаж байгаа эсэхийг шалгана.
5. IDE-г бүрэн хааж дахин нээгээд `mongolgpt`-г нэгдсэн терминалаас дахин ажиллуулна.

Асуудал үргэлжилбэл [алдаа шийдэх зааврын](../troubleshooting/) дагуу version, log, үйлдлийн системийн мэдээллээ цуглуулна.
