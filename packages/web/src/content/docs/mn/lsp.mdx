MongolGPT нь хэлний серверийн протокол (LSP) серверүүдтэй нэгдэж, оношлогоог агентын санал хүсэлт болгон ашиглах боломжтой.

---

LSP нь анхдагчаар идэвхгүй байна. Идэвхжүүлсэн үед дээрх файлын өргөтгөлүүдийн аль нэгийг илрүүлж, шаардлага хангасан үед серверүүд ажиллаж эхэлнэ.

:::note
Та `MONGOLGPT_DISABLE_LSP_DOWNLOAD` орчны хувьсагчийг `true` болгож тохируулснаар LSP серверийн автомат таталтыг идэвхгүй болгож болно.
:::

---

## Энэ хэрхэн ажилладаг вэ

LSP идэвхжсэн бөгөөд mongolgpt файлыг нээх үед энэ нь:

1. Файлын өргөтгөлийг идэвхжүүлсэн бүх LSP сервертэй харьцуулна.
2. Ажиллаагүй бол тохирох LSP серверийг эхлүүлнэ.

---

## Шилдэг туршлага

LSP хэлний серверүүдээс оношилгоог хийснээр агентад асуудлыг олж засварлахад тусална. Энэ нь зарим төслүүдэд ашигтай боловч үргэлж эерэг үр дүн байдаггүй.

Хэлний серверүүд синхрончлолгүй болж, их хэмжээний санах ой ашиглаж, хувилбар эсвэл төслөөс хамаарч өөр өөр байж, агентын ажлын урсгалыг удаашруулж болно. Олон төслүүдэд агент нь lint, typecheck эсвэл бусад оношилгооны CLI хэрэгслийг шууд ажиллуулах нь илүү дээр байдаг тул алдааг агентын давталт руу буцаан оруулдаг. Эдгээр тушаалуудыг `AGENTS.md` эсвэл ур чадвар гэх мэт зааварчилгааны файлд баримтжуулж, агент юу ажиллуулахаа мэдэх болно. Таны төсөл нэмэлт хэлний серверийн санал хүсэлтээс ашиг хүртэх үед LSP-г идэвхжүүлнэ үү.

---

## Тохируулах

Та өөрийн mongolgpt тохиргооны `lsp` хэсгээс LSP серверүүдийг идэвхжүүлж, өөрчлөх боломжтой.

Бүх суурилуулсан LSP серверүүдийг идэвхжүүлэхийн тулд `lsp`-г `true` гэж тохируулна уу.

```json title="mongolgpt.json"
{
  "$schema": "https://mongolgpt.duckdns.org/config.json",
  "lsp": true
}
```

Хүчингүй болгох эсвэл захиалгат серверүүдийг тохируулахдаа суулгасан програмуудыг идэвхжүүлэхийн тулд объект ашиглана уу.

```json title="mongolgpt.json"
{
  "$schema": "https://mongolgpt.duckdns.org/config.json",
  "lsp": {}
}
```

Тохируулсан LSP серверийн оролт бүр дараахь зүйлийг дэмждэг:

Зөвхөн серверийг идэвхгүй болгохоос бусад тохиолдолд серверийн оруулгад `command` хэрэгтэй.

| Үл хөдлөх хөрөнгө | Төрөл | Тодорхойлолт |
| ---------------- | -------- | ------------------------------------------------- |
| `disabled` | логикийн | LSP серверийг идэвхгүй болгохын тулд үүнийг `true` гэж тохируулна уу |
| `command` | мөр[] | LSP серверийг эхлүүлэх команд |
| `extensions` | мөр[] | Энэ LSP серверийн ажиллах ёстой файлын өргөтгөлүүд |
| `env` | объект | Серверийг эхлүүлэх үед тохируулах орчны хувьсагчууд |
| `initialization` | объект | LSP сервер рүү илгээх эхлүүлэх сонголтууд |

Зарим жишээг харцгаая.

---

### Хүрээлэн буй орчны хувьсагчид

LSP серверийг эхлүүлэх үед орчны хувьсагчдыг тохируулахын тулд `env` шинж чанарыг ашиглана уу:

```json title="mongolgpt.json" {5-8}
{
  "$schema": "https://mongolgpt.duckdns.org/config.json",
  "lsp": {
    "rust": {
      "command": ["rust-analyzer"],
      "env": {
        "RUST_LOG": "debug"
      }
    }
  }
}
```

---

### Эхлүүлэх сонголтууд

`initialization` сервер рүү эхлүүлэх сонголтыг дамжуулахын тулд `initialization` шинж чанарыг ашиглана уу. Эдгээр нь LSP `initialize` хүсэлтийн үед илгээсэн серверийн тусгай тохиргоонууд юм:

```json title="mongolgpt.json" {5-13}
{
  "$schema": "https://mongolgpt.duckdns.org/config.json",
  "lsp": {
    "custom-lsp": {
      "command": ["custom-lsp-server", "--stdio"],
      "extensions": [".custom"],
      "initialization": {
        "preferences": {
          "importModuleSpecifierPreference": "relative"
        }
      }
    }
  }
}
```

:::note
Эхлүүлэх сонголтууд LSP серверээс хамаарч өөр өөр байдаг. Боломжтой сонголтуудыг LSP серверийнхээ бичиг баримтаас шалгана уу.
:::

---

### LSP серверүүдийг идэвхгүй болгож байна

Хэрэв `lsp`-г орхигдуулсан бол бүх LSP сервер идэвхгүй болно. Өөр тохиргоог идэвхжүүлсний дараа бүх LSP серверүүдийг идэвхгүй болгохын тулд `lsp`-г `false` гэж тохируулна уу:

```json title="mongolgpt.json" {3}
{
  "$schema": "https://mongolgpt.duckdns.org/config.json",
  "lsp": false
}
```

**тодорхой** LSP серверийг идэвхгүй болгохын тулд `disabled`-г `true` гэж тохируулна уу:

```json title="mongolgpt.json" {5}
{
  "$schema": "https://mongolgpt.duckdns.org/config.json",
  "lsp": {
    "typescript": {
      "disabled": true
    }
  }
}
```

---

### Захиалгат LSP серверүүд

Та тушаал болон файлын өргөтгөлүүдийг зааж өгснөөр захиалгат LSP серверүүдийг нэмж болно:

```json title="mongolgpt.json" {4-7}
{
  "$schema": "https://mongolgpt.duckdns.org/config.json",
  "lsp": {
    "custom-lsp": {
      "command": ["custom-lsp-server", "--stdio"],
      "extensions": [".custom"]
    }
  }
}
```

## Нэмэлт мэдээлэл

### PHP Intelephense

PHP Intelephense лицензийн түлхүүрээр дамжуулан дээд зэргийн функцуудыг санал болгодог. Та лицензийн түлхүүрийг дараах хаягаар текст файлд байрлуулснаар (зөвхөн) өгөх боломжтой.

- macOS/Linux дээр: `$HOME/intelephense/license.txt`
- Windows дээр: `%USERPROFILE%/intelephense/license.txt`

Файл нь нэмэлт агуулгагүй зөвхөн лицензийн түлхүүрийг агуулсан байх ёстой.
