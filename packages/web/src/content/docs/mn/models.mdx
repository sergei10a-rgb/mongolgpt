---

MongolGPT нь **75+ LLM үйлчилгээ үзүүлэгчдийг** дэмжихийн тулд [AI SDK](https://ai-sdk.dev/) болон [Models.dev](https://models.dev) ашигладаг бөгөөд орон нутгийн загваруудыг ажиллуулахыг дэмждэг.

---

## Үйлчилгээ үзүүлэгчид

Ихэнх алдартай үйлчилгээ үзүүлэгчид анхдагчаар урьдчилан ачаалагдсан байдаг. Хэрэв та `/connect` тушаалаар үйлчилгээ үзүүлэгчийн итгэмжлэлүүдийг нэмсэн бол MongolGPT-г эхлүүлэх үед тэдгээр нь бэлэн болно.

[providers](/docs/providers)-ийн талаар нэмэлт мэдээлэл аваарай.

---

## Загвар сонгоно уу

Үйлчилгээ үзүүлэгчээ тохируулсны дараа та дараахийг бичээд хүссэн загвараа сонгох боломжтой.

```bash frame="none"
/models
```

---

## Санал болгож буй загварууд

Долоо хоног бүр шинэ загварууд гарч ирдэг маш олон загварууд байдаг.

:::tip
Бидний санал болгож буй загваруудын аль нэгийг ашиглах талаар бодож үзээрэй.
:::

Гэсэн хэдий ч, тэдгээрийн зөвхөн цөөн хэдэн нь код үүсгэх, багажийн дуудлага хийх чадвартай байдаг.

Тодорхой дараалалгүйгээр MongolGPT-тэй сайн ажилладаг хэд хэдэн загвар энд байна. (Энэ бол бүрэн жагсаалт биш бөгөөд заавал шинэчлэгдсэн байх албагүй):

- GPT 5.2
- GPT 5.1 Кодекс
- Claude Opus 4.5
- Claude Sonnet 4.5
- Minimax M2.1
- Gemini 3 Pro

---

## Өгөгдмөл тохируул

Эдгээрийн аль нэгийг өгөгдмөл загвар болгон тохируулахын тулд та өөрийн тохиргоонд `model` товчлуурыг тохируулж болно
MongolGPT тохиргоо.

```json title="mongolgpt.json" {3}
{
  "$schema": "https://mgpt.mn/config.json",
  "model": "lmstudio/google/gemma-3n-e4b"
}
```

Энд бүтэн ID нь `provider_id/model_id` байна. Жишээлбэл, OpenRouter дээрх Gemini 2.5 Flash загварыг `openrouter/google/gemini-2.5-flash` гэж заана. Free Auto ашиглах бол [бүртгэл ба багцын зааврыг](/docs/account) үзнэ үү.

Хэрэв та [захиалгат үйлчилгээ үзүүлэгч](/docs/providers#custom)-г тохируулсан бол `provider_id` нь таны тохиргооны `provider` хэсгийн түлхүүр бөгөөд `model_id` нь `provider.models`-ийн түлхүүр юм.

## Загваруудыг тохируулах

Та тохиргоогоор дамжуулан загварын сонголтыг дэлхийн хэмжээнд тохируулах боломжтой.

```jsonc title="mongolgpt.jsonc" {7-12,19-24}
{
  "$schema": "https://mgpt.mn/config.json",
  "provider": {
    "openai": {
      "models": {
        "gpt-5": {
          "options": {
            "reasoningEffort": "high",
            "textVerbosity": "low",
            "reasoningSummary": "auto",
            "include": ["reasoning.encrypted_content"],
          },
        },
      },
    },
    "anthropic": {
      "models": {
        "claude-sonnet-4-5-20250929": {
          "options": {
            "thinking": {
              "type": "enabled",
              "budgetTokens": 16000,
            },
          },
        },
      },
    },
  },
}
```

Энд бид хоёр суурилуулсан загварт зориулсан глобал тохиргоог хийж байна: `openai` үйлчилгээ үзүүлэгчээр хандах үед `gpt-5`, `anthropic` үйлчилгээ үзүүлэгчээр дамжуулан хандах үед `claude-sonnet-4-20250514`.
Баригдсан үйлчилгээ үзүүлэгч болон загварын нэрийг [Models.dev](https://models.dev) дээрээс олж болно.

Та мөн ашиглаж байгаа аливаа агентдаа эдгээр тохируулгыг тохируулах боломжтой. Агентын тохиргоо нь энд байгаа бүх глобал сонголтыг дарна. [Илүү ихийг мэдэж аваарай](/docs/agents/#additional).

Та мөн суулгасан хувилбаруудыг өргөтгөх захиалгат хувилбаруудыг тодорхойлж болно. Хувилбарууд нь давхардсан оруулгууд үүсгэхгүйгээр ижил загварын өөр өөр тохиргоог тохируулах боломжийг танд олгоно.

```jsonc title="mongolgpt.jsonc" {6-21}
{
  "$schema": "https://mgpt.mn/config.json",
  "provider": {
    "mongolgpt": {
      "models": {
        "gpt-5": {
          "variants": {
            "high": {
              "reasoningEffort": "high",
              "textVerbosity": "low",
              "reasoningSummary": "auto",
            },
            "low": {
              "reasoningEffort": "low",
              "textVerbosity": "low",
              "reasoningSummary": "auto",
            },
          },
        },
      },
    },
  },
}
```

## Хувилбарууд

Олон загварууд өөр өөр тохиргоотой олон хувилбаруудыг дэмждэг. MongolGPT алдартай үйлчилгээ үзүүлэгчдэд зориулсан анхдагч хувилбаруудыг нийлүүлдэг.

### Баригдсан хувилбарууд

MongolGPT нь олон үйлчилгээ үзүүлэгчийн үндсэн хувилбаруудыг нийлүүлдэг:

**Anthropic**:

- `high` - Өндөр бодолтой төсөв (анхдагч)
- `max` - Хамгийн их бодох төсөв

**OpenAI**:

Загвараас хамааран өөр өөр байдаг боловч ойролцоогоор:

- one` - Ямар ч үндэслэлгүй
- `minimal` - Хамгийн бага үндэслэл гаргах хүчин чармайлт
- `low` - Бодол санааны хүчин чармайлт бага
- `medium` - Дунд зэргийн үндэслэл гаргах хүчин чармайлт
- `high` - Өндөр үндэслэл гаргах хүчин чармайлт
- `xhigh` - Хэт өндөр үндэслэл гаргах хүчин чармайлт

**Google**:

- `low` - Бага хүчин чармайлт / жетон төсөв
- `high` - Өндөр хүчин чармайлт/жетон төсөв

:::tip
Энэ жагсаалт нь цогц биш юм. Бусад олон үйлчилгээ үзүүлэгчид мөн анхдагч тохиргоотой байдаг.
:::

### Захиалгат хувилбарууд

Та одоо байгаа хувилбаруудыг хүчингүй болгох эсвэл өөрийн хувилбарыг нэмж болно:

```jsonc title="mongolgpt.jsonc" {7-18}
{
  "$schema": "https://mgpt.mn/config.json",
  "provider": {
    "openai": {
      "models": {
        "gpt-5": {
          "variants": {
            "thinking": {
              "reasoningEffort": "high",
              "textVerbosity": "low",
            },
            "fast": {
              "disabled": true,
            },
          },
        },
      },
    },
  },
}
```

### Циклийн хувилбарууд

Хувилбаруудын хооронд хурдан шилжихийн тулд `variant_cycle` товчлуурыг ашиглана уу. [Илүү ихийг мэдэж аваарай](/docs/keybinds).

## Загваруудыг ачаалж байна

MongolGPT эхлэх үед дараах тэргүүлэх дарааллаар загваруудыг шалгана:

1. `--model` эсвэл `-m` тушаалын мөр. Формат нь тохиргооны файлтай адил байна: `provider_id/model_id`.

2. MongolGPT тохиргооны загварын жагсаалт.

   ```json title="mongolgpt.json"
   {
     "$schema": "https://mgpt.mn/config.json",
     "model": "anthropic/claude-sonnet-4-20250514"
   }
   ```

   Энд байгаа формат нь `provider/model`.

3. Хамгийн сүүлд ашигласан загвар.

4. Дотоод давуу талыг ашигласан анхны загвар.
