## Дэвсгэр

Нээлттэй загварууд үнэхээр сайн болсон. Тэд одоо гүйцэтгэлд ойрхон байна
кодлох даалгаврын өмчийн загварууд. Мөн олон үйлчилгээ үзүүлэгчид тэдэнд үйлчлэх боломжтой учраас
өрсөлдөх чадвартай, тэд ихэвчлэн хамаагүй хямд байдаг.

Гэсэн хэдий ч найдвартай, бага хоцрогдолтой хандах нь хэцүү байж болно. Үйлчилгээ үзүүлэгчид
чанар, хүртээмжээрээ ялгаатай.

:::tip
Бид MongolGPT-тэй сайн ажилладаг загвар, үйлчилгээ үзүүлэгчдийн сонгогдсон бүлгийг туршиж үзсэн.
:::

Үүнийг засахын тулд бид хэд хэдэн зүйлийг хийсэн:

1. Бид сонгогдсон нээлттэй загвар өмсөгчдийг туршиж үзээд багийнхантай нь хэрхэн хийх талаар ярилцсан
   тэднийг ажиллуулах нь дээр.
2. Дараа нь бид эдгээр үйлчилгээ үзүүлж байгаа эсэхийг шалгахын тулд цөөн хэдэн үйлчилгээ үзүүлэгчтэй ажилласан
   зөв.
3. Эцэст нь бид загвар/үйлчилгээ үзүүлэгчийн хослолыг харьцуулж үзээд гарч ирлээ
   Бидний санал болгож буй жагсаалтын хамт.

MongolGPT Go нь эдгээр загваруудыг эхний сард **$5, дараа нь **сард 10$** үнээр авах боломжийг олгоно.

## Энэ хэрхэн ажилладаг

MongolGPT Go нь MongolGPT дээрх бусад үйлчилгээ үзүүлэгчтэй адил ажилладаг.

1. Та **<a href={console}>MongolGPT Zen</a>**-д нэвтэрч, Go-д бүртгүүлж, мөн
   API түлхүүрээ хуулна уу.
2. Та `/connect` командыг TUI-д ажиллуулж, `MongolGPT Go`-г сонгоод буулгана уу.
   таны API түлхүүр.
3. TUI дээр `/models` гэж ажиллуулж Go-ээр дамжуулан авах боломжтой загваруудын жагсаалтыг харна уу.

Нэг ажлын талбарт зөвхөн нэг гишүүн MongolGPT Go-д бүртгүүлэх боломжтой.
:::

## Зорилго

Бид MongolGPT үүсгэсэн:

1. AI кодыг хямд үнээр илүү олон хүнд **хүртээмжтэй** болго.
2. Шилдэг нээлттэй кодчилол загваруудад **найдвартай** хандах боломжийг олгох.
3. Кодлох агентын хэрэглээнд зориулж **туршиж, харьцуулсан** загваруудыг сонго.
4. MongolGPT бүхий өөр үйлчилгээ үзүүлэгчийг ашиглахыг зөвшөөрснөөр **түгжихгүй** байх.
