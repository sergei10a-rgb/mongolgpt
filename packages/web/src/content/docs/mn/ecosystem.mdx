---

MongolGPT-ийн экосистемийн хуудсанд одоогоор зөвхөн баталгаажсан холболтын төрлүүдийг харуулна. Албан ёсны зах зээл болон багцын бүртгэл бэлэн болоогүй тул гуравдагч талын дахин брэндчилсэн багцыг энд жагсаахгүй.

## Баталгаатай холболтын төрлүүд

| Төрөл              | Одоогийн төлөв                                                                                                               |
| ------------------ | ---------------------------------------------------------------------------------------------------------------------------- |
| MCP сервер         | `mongolgpt compat import plan/apply` тушаал болон ширээний аппын холболтын хөтчөөр тушаал, дотоод зам, URL, тохиргоог танина |
| Чадвар             | Claude, Codex, Goose маягийн чадварын сан болон тохиргоог таних суурь импортлогчтой                                          |
| Нэмэлт             | Дотоод нэмэлтийн файл болон хувийн npm багцыг тохиргоонд зааж ажиллуулна                                                     |
| Үйлчилгээ үзүүлэгч | MongolGPT бүртгэлээр нэвтрэх, API түлхүүр, дотоод болон үйлчилгээ үзүүлэгчийн тохиргооны холболтыг дэмжинэ                   |

## Импортын ажлын урсгал

```bash
mongolgpt compat import plan "npx -y @modelcontextprotocol/server-filesystem C:\\Users\\me"
mongolgpt compat import apply "npx -y @modelcontextprotocol/server-filesystem C:\\Users\\me"
```

Ширээний аппын `Тохиргоо -> Холболт` хэсэгт тушаал, зам, URL эсвэл тохиргоо оруулж шалгаад нэмнэ.

## Нийтлэхээс өмнөх дүрэм

- Нийтлэгдээгүй npm, Homebrew, AUR, Chocolatey эсвэл Scoop багцыг албан ёсны суулгац гэж харуулахгүй.
- Гуравдагч талын нэмэлтийг MongolGPT нэрээр дахин брэндчилж, бодитоор байхгүй URL эсвэл багц руу заахгүй.
- Шинэ холболтыг эхлээд импортлогч болон тохируулагчаар шалгана. Албан ёсны бүртгэл эсвэл зах зээлд нийтэлсний дараа баримт бичигт суулгах тушаалыг нэмнэ.
