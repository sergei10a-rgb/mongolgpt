Лавлагаа нь MongolGPT-д одоогийн төслийн гаднах лавлах руу хандах боломжийг олгоно. Ажиллаж байхдаа баримт бичиг, хуваалцсан номын сан, жишээнүүд эсвэл өөр хадгалах газрыг ашиглах боломжтой болгохын тулд тэдгээрийг ашиглана уу.

Лавлагаа нь `mongolgpt.json` эсвэл `mongolgpt.jsonc` дахь өөр нэрээр тохируулагдсан.

```jsonc title="mongolgpt.jsonc"
{
  "$schema": "https://mongolgpt.duckdns.org/config.json",
  "references": {
    "docs": {
      "path": "../product-docs",
      "description": "Use for product behavior and documentation conventions",
    },
    "sdk": {
      "repository": "sergei10a-rgb/mongolgpt-sdk-js",
      "branch": "main",
      "description": "Use for JavaScript SDK implementation details",
    },
  },
}
```

## Орон нутгийн лавлахууд

Орон нутгийн лавлахыг лавлахын тулд `path` ашиглана уу.

```jsonc title="mongolgpt.jsonc"
{
  "references": {
    "docs": {
      "path": "../docs",
    },
  },
}
```

Замууд нь дараахь байж болно.

- Лавлагааг тодорхойлсон тохиргооны файлтай холбоотой
- Үнэмлэхүй, тухайлбал `/home/user/docs`
- `~/docs` гэх мэт таны гэрийн лавлахтай холбоотой

Та мөн мөрийн товчлолыг ашиглаж болно:

```jsonc title="mongolgpt.jsonc"
{
  "references": {
    "docs": "../docs",
  },
}
```

## Git репозиторууд

Git репозиторыг лавлахдаа `repository` ашиглана уу. MongolGPT репозиторыг локал репозиторын кэшдээ материалжуулж, шалгагдсан эх сурвалжийг лавлах лавлах болгон ашиглах боломжтой болгоно.

```jsonc title="mongolgpt.jsonc"
{
  "references": {
    "effect": {
      "repository": "Effect-TS/effect",
      "branch": "main",
    },
  },
}
```

`repository` Git URLs, хост/замын лавлагаа, GitHub `owner/repo` товчлолыг хүлээн зөвшөөрдөг. Нэмэлт `branch` талбар нь салбар эсвэл лавлахыг сонгоно. `branch` байхгүй бол MongolGPT репозиторын анхдагч салбарыг ашигладаг.

Салбар, тайлбар болон бусад сонголт шаардлагагүй үед та мөрийн товчлолыг ашиглаж болно:

```jsonc title="mongolgpt.jsonc"
{
  "references": {
    "effect": "Effect-TS/effect",
  },
}
```

:::note
Git лавлагааг асинхрон байдлаар шинэчилдэг. Шинээр тохируулсан репозитор клончлох эсвэл шинэчлэхэд хэсэг хугацаа зарцуулагдаж магадгүй.
:::

## Хэрэглээг тайлбарлана уу

Агент хэзээ лавлагаа ашиглах ёстойг тайлбарлахын тулд `description` нэмнэ үү.

```jsonc title="mongolgpt.jsonc"
{
  "references": {
    "design-system": {
      "path": "../design-system",
      "description": "Use when implementing UI components or design tokens",
    },
  },
}
```

MongolGPT нь агентийн контекст дэх тайлбар бүхий лавлагааг агуулдаг. Тодорхойлолт нь ижил төстэй агуулгатай лавлагааг ялгахад хангалттай богино бөгөөд тодорхой байх ёстой. Тодорхойлолтгүй лавлагааг автоматаар бөглөх, шууд ашиглах замаар ашиглах боломжтой боловч агентуудад сурталчлахгүй.

## Автоматаар бөглөх бичилтүүдийг нуу

`hidden`-д `@` автомат бөглөхөөс лавлагааг орхихын тулд `true` гэж тохируулна уу.

```jsonc title="mongolgpt.jsonc"
{
  "references": {
    "internal": {
      "path": "../internal",
      "description": "Use for internal implementation details",
      "hidden": true,
    },
  },
}
```

`hidden` зөвхөн автоматаар бөглөхөд нөлөөлдөг. Тодорхойлолт бүхий далд лавлагааг агентийн контекст оруулсан хэвээр байна.

## Лавлагаа ашиглах

Тохируулсан лавлагаа нь TUI `@` автомат бөглөх хэсэгт харагдана. Лавлагааны язгуурыг хавсаргах бол `@alias`, доторх файлуудыг хайх бол `@alias/` гэж бичнэ үү.

```text
Compare this implementation with @sdk/src/client.ts
```

Агентууд мөн өөрсдийн системийн контекст дэх тайлбар бүхий тохируулсан лавлагааны шийдвэрлэсэн замууд болон тайлбаруудыг хүлээн авдаг бөгөөд ингэснээр тэд лавлагааг холбогдох үед нь гараар хавсаргахгүйгээр шалгаж болно.

MongolGPT нь гадаад лавлах зөвшөөрлийн хилээр дамжуулан лавлагаа лавлахыг автоматаар зөвшөөрдөг. Хэрэгслийн ердийн зөвшөөрөл хэрэгжсэн хэвээр байна; жишээлбэл, лавлах болгон тохируулсан лавлах учраас файл засварлах боломжгүй агент засварлах эрх олж авдаггүй.

## Талбаруудыг тохируулах

| Талбай        | Орон нутгийн | Git  | Тодорхойлолт                                      |
| ------------- | ------------ | ---- | ------------------------------------------------- |
| `path`        | Тийм         | Үгүй | Орон нутгийн лавлах лавлах                        |
| `repository`  | Үгүй         | Тийм | Git URL, хост/зам, эсвэл GitHub `owner/repo` утга |
| `branch`      | Үгүй         | Тийм | Нэмэлт Git салбар эсвэл ref                       |
| `description` | Тийм         | Тийм | Лавлагааг хэзээ ашиглахыг тайлбарласан заавар     |
| `hidden`      | Тийм         | Тийм | Лавлагааг TUI `@` автомат бөглөхөөс нуу           |

Лавлах нэр нь хоосон байж болохгүй эсвэл `/`, хоосон зай, арын тэмдэг, таслал агуулсан байж болохгүй.
