MongolGPT нь тухайн хэлний форматлагч ашиглан файлуудыг бичиж эсвэл зассаны дараа форматлах боломжтой. Форматлагч нь анхдагчаар идэвхгүй болсон; MongolGPT тэдгээрийг ажиллуулахаас өмнө тохиргоондоо идэвхжүүлнэ үү.

---

## Баригдсан

MongolGPT алдартай хэл болон фреймворкуудад зориулсан хэд хэдэн суулгасан форматлагчтай ирдэг. Доорх нь форматлагчдын жагсаалт, дэмжигдсэн файлын өргөтгөлүүд, түүнд шаардлагатай командууд эсвэл тохиргооны сонголтуудын жагсаалт юм.

## Энэ хэрхэн ажилладаг

MongolGPT файл бичих, засварлах, форматлагчдыг идэвхжүүлсэн үед энэ нь:

1. Бүх идэвхжүүлсэн форматлагчтай файлын өргөтгөлийг шалгана.
2. Файл дээр тохирох форматлагч командыг ажиллуулна.
3. Форматын өөрчлөлтийг хэрэгжүүлнэ.

Идэвхжүүлсэн форматлагчийн хувьд энэ процесс арын дэвсгэр дээр явагдана.

---

## Тохируулах

Та өөрийн MongolGPT тохиргооны `formatter` хэсгээс форматлагчдыг идэвхжүүлж, өөрчлөх боломжтой.

Бүх суулгасан форматлагчдыг идэвхжүүлэхийн тулд `formatter`-г `true` гэж тохируулна уу.

```json title="mongolgpt.json"
{
  "$schema": "https://mongolgpt.duckdns.org/config.json",
  "formatter": true
}
```

Хүчингүй болгох эсвэл өөрчлөн форматлагчийг тохируулахын тулд суулгацыг идэвхжүүлэхийн тулд объект ашиглана уу.

```json title="mongolgpt.json"
{
  "$schema": "https://mongolgpt.duckdns.org/config.json",
  "formatter": {}
}
```

Форматлагчийн тохиргоо бүр дараахь зүйлийг дэмждэг.

| Үл хөдлөх хөрөнгө | Төрөл | Тодорхойлолт |
| ------------- | -------- | ------------------------------------------------------------------------------------------ |
| `disabled` | логикийн | Форматлагчийг идэвхгүй болгохын тулд үүнийг `true` гэж тохируулна уу |
| `command` | мөр[] | Форматлахын тулд ажиллуулах тушаал. Захиалгат форматлагчдад шаардлагатай; суулгацын хувьд сонголттой. |
| `environment` | объект | Форматлагчийг ажиллуулах үед тохируулах орчны хувьсагчууд |
| `extensions` | мөр[] | Энэ форматлагчийн ажиллах ёстой файлын өргөтгөлүүд |

Зарим жишээг харцгаая.

---

### Форматлагчдыг идэвхгүй болгож байна

Хэрэв `formatter`-г орхигдуулсан бол бүх форматлагч идэвхгүй болно. Өөр тохиргоог идэвхжүүлсний дараа бүх форматлагчдыг идэвхгүй болгохын тулд `formatter`-г `false` гэж тохируулна уу:

```json title="mongolgpt.json" {3}
{
  "$schema": "https://mongolgpt.duckdns.org/config.json",
  "formatter": false
}
```

**Тодорхой** форматлагчийг идэвхгүй болгохын тулд `disabled`-г `true` болгож тохируулна уу:

```json title="mongolgpt.json" {5}
{
  "$schema": "https://mongolgpt.duckdns.org/config.json",
  "formatter": {
    "prettier": {
      "disabled": true
    }
  }
}
```

---

### Захиалгат форматлагч

Та `environment` эсвэл `extensions` гэх мэт сонголтоор суурилуулсан форматлагчдыг тохируулах боломжтой. Захиалгат форматлагч нэмэхийн тулд `command` болон `extensions`-г зааж өгнө үү:

```json title="mongolgpt.json" {4-14}
{
  "$schema": "https://mongolgpt.duckdns.org/config.json",
  "formatter": {
    "prettier": {
      "command": ["npx", "prettier", "--write", "$FILE"],
      "environment": {
        "NODE_ENV": "development"
      },
      "extensions": [".js", ".ts", ".jsx", ".tsx"]
    },
    "custom-markdown-formatter": {
      "command": ["deno", "fmt", "$FILE"],
      "extensions": [".md"]
    }
  }
}
```

Тушаал дахь **`$FILE` орлуулагч** нь форматлаж буй файлын замаар солигдох болно.
