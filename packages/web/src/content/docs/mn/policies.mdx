---
title: Бодлогууд
description: MongolGPT аль тохируулсан нөөцүүдийг ашиглаж болохыг хянах.
---

Бодлого нь MongolGPT нэртэй нөөц дээр үйлдэл хийж болох эсэхийг хянадаг. Энэ функц нь туршилтын шинж чанартай бөгөөд `mongolgpt.json` дахь `experimental.policies` массиваар тохируулагдсан.

Бодлого нь [permissions](/docs/permissions)-аас тусдаа. Зөвшөөрөл нь сессийн үед ямар хэрэглүүр хийж болохыг хянадаг бол бодлого нь MongolGPT үйлчилгээ үзүүлэгч гэх мэт нөөцийг ашиглаж болох эсэхийг хянадаг.

---

## Тохиргоо

Бодлогын мэдэгдэл бүр гурван талбартай:

- `effect` - `"allow"` эсвэл `"deny"`.
- `action` - Хянаж буй үйл ажиллагаа.
- `resource` - Нөөц ID эсвэл мэдэгдэлд хамаарах орлуулагч хэв маяг.

Жишээ нь, `openai` үйлчилгээ үзүүлэгчийг ашиглахаас татгалзах:

```json title="mongolgpt.json"
{
  "$schema": "https://mgpt.mn/config.json",
  "experimental": {
    "policies": [
      {
        "effect": "deny",
        "action": "provider.use",
        "resource": "openai"
      }
    ]
  }
}
```

Бодлогын дагуу татгалзсан үйлчилгээ үзүүлэгч нь итгэмжлэлтэй эсвэл зөв тохируулагдсан байсан ч загвар сонгох эсвэл загвар ашиглах боломжгүй.

---

## Боломжтой бодлого

MongolGPT одоогоор нэг бодлогын арга хэмжээг дэмждэг:

| Үйлдэл         | Нөөц                                     | Тодорхойлолт                                                   |
| -------------- | ---------------------------------------- | -------------------------------------------------------------- |
| `provider.use` | Үйлчилгээ үзүүлэгч ID, жишээ нь `openai` | LLM үйлчилгээ үзүүлэгчийг ашиглахыг зөвшөөрөх эсвэл татгалзах. |

Цаашид бодлогын олон арга хэмжээ нэмж болно.

---

## Тохирох

`resource` талбар нь орлуулагч тэмдэгт тохирохыг дэмждэг. `*` тэг ба түүнээс дээш тэмдэгттэй тааруулах бол `?` нэг тэмдэгтийг тааруулна.

```json title="mongolgpt.json"
{
  "$schema": "https://mgpt.mn/config.json",
  "experimental": {
    "policies": [
      {
        "effect": "deny",
        "action": "provider.use",
        "resource": "company-*"
      }
    ]
  }
}
```

Энэ нь `company-us` болон `company-eu` зэрэг үйлчилгээ үзүүлэгчдийг үгүйсгэдэг.

---

## Дүрэм

Олон хэллэг таарч байвал хамгийн сүүлд тохирох мэдэгдэл ялна. Эхлээд өргөн хүрээтэй дүрмийг, дараа нь илүү тодорхой үл хамаарах зүйлийг тавь.

Жишээлбэл, зөвхөн Anthropic-г зөвшөөрөх:

```json title="mongolgpt.json"
{
  "$schema": "https://mgpt.mn/config.json",
  "experimental": {
    "policies": [
      {
        "effect": "deny",
        "action": "provider.use",
        "resource": "*"
      },
      {
        "effect": "allow",
        "action": "provider.use",
        "resource": "anthropic"
      }
    ]
  }
}
```

Хэрэв үйлчилгээ үзүүлэгчтэй тохирох бодлого байхгүй бол үйлчилгээ үзүүлэгчийн хэрэглээг өгөгдмөлөөр зөвшөөрнө.

Бодлогуудыг таны глобал тохиргоо болон төслийн тохиргоонд тохируулж болно. Хэрэв хоёр байршлын бодлого нь ижил үйлчилгээ үзүүлэгчтэй таарч байвал таны дэлхийн бодлого төслийн бодлогоос давуу эрхтэй болно. Энэ нь таны дэлхий даяар үгүйсгэсэн үйлчилгээ үзүүлэгчийг хадгалах газар дахин идэвхжүүлэхээс сэргийлнэ.

---

## Үйлчилгээ үзүүлэгчдийн жагсаалт

Үйлчилгээ үзүүлэгчийн хандалтыг хянахдаа хуучин `disabled_providers` болон `enabled_providers` тохиргооны оронд бодлогыг ашиглаарай.

`disabled_providers`-г солихын тулд:

```json title="mongolgpt.json"
{
  "experimental": {
    "policies": [
      { "effect": "deny", "action": "provider.use", "resource": "openai" },
      { "effect": "deny", "action": "provider.use", "resource": "google" }
    ]
  }
}
```

`enabled_providers`-г солихын тулд эхлээд бүх үйлчилгээ үзүүлэгчээс татгалзаж, дараа нь сонгосон үйлчилгээ үзүүлэгчдийг зөвшөөрнө үү:

```json title="mongolgpt.json"
{
  "experimental": {
    "policies": [
      { "effect": "deny", "action": "provider.use", "resource": "*" },
      { "effect": "allow", "action": "provider.use", "resource": "anthropic" },
      { "effect": "allow", "action": "provider.use", "resource": "openai" }
    ]
  }
}
```
