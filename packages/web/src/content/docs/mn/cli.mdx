Анхдагчаар MongolGPT CLI нь ямар ч аргументгүйгээр ажиллах үед [TUI](/docs/tui)-г эхлүүлдэг.

```bash
mongolgpt
```

Гэхдээ энэ хуудсан дээр бичигдсэн тушаалуудыг бас хүлээн авдаг. Энэ нь MongolGPT-тай программчлан харилцах боломжийг олгоно.

```bash
mongolgpt run "JavaScript-ийн closure хэрхэн ажилладгийг тайлбарла"
```

---

### туй

MongolGPT терминалын хэрэглэгчийн интерфэйсийг эхлүүлнэ үү.

```bash
mongolgpt [project]
```

#### Тугнууд

| Туг                                         | Богино | Тодорхойлолт                                                                  |
| ------------------------------------------- | ------ | ----------------------------------------------------------------------------- |
| <nobr><code>{"--continue"}</code></nobr>    | `-c`   | Сүүлийн сессийг үргэлжлүүлэх                                                  |
| <nobr><code>{"--session"}</code></nobr>     | `-s`   | Үргэлжлүүлэхийн тулд сесс ID                                                  |
| <nobr><code>{"--fork"}</code></nobr>        |        | Үргэлжлүүлэхдээ сессийг хойшлуул (`--continue` эсвэл `--session` ашиглана уу) |
| <nobr><code>{"--prompt"}</code></nobr>      |        | Ашиглахыг сануулах                                                            |
| <nobr><code>{"--model"}</code></nobr>       | `-m`   | Үйлчилгээ үзүүлэгч/загвар хэлбэрээр ашиглах загвар                            |
| <nobr><code>{"--agent"}</code></nobr>       |        | Ашиглах агент                                                                 |
| <nobr><code>{"--port"}</code></nobr>        |        | Сонсох порт                                                                   |
| <nobr><code>{"--hostname"}</code></nobr>    |        | Сонсох хостын нэр                                                             |
| <nobr><code>{"--mdns"}</code></nobr>        |        | mDNS илрүүлэлтийг идэвхжүүлэх                                                 |
| <nobr><code>{"--mdns-domain"}</code></nobr> |        | Захиалгат mDNS домэйн нэр                                                     |
| <nobr><code>{"--cors"}</code></nobr>        |        | CORS \| зөвшөөрөх нэмэлт хөтчийн гарал үүсэл(үүд).                            |

## тушаалууд

MongolGPT CLI нь мөн дараах тушаалуудтай.

---

### төлөөлөгч

MongolGPT-н агентуудыг удирдах.

```bash
mongolgpt agent [command]
```

---

#### үүсгэх

Өөрчлөн тохируулсан шинэ агент үүсгэнэ үү.

```bash
mongolgpt agent create
```

Энэ команд нь тусгай системийн хүсэлт, зөвшөөрлийн тохиргоо бүхий шинэ агент үүсгэхэд тань туслах болно. Таны зөвшөөрөөгүй бүх зүйлийг үүсгэсэн агентын нүүрэнд үгүйсгэдэг.

#### Тугнууд

| Туг                                         | Богино | Тодорхойлолт                                                                                                                                                                                                                                |
| ------------------------------------------- | ------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| <nobr><code>{"--path"}</code></nobr>        |        | Агент файлыг бичих лавлах (өгөгдмөл нь глобал юмуу `.mongolgpt/agent` сануулгыг үндэслэн)                                                                                                                                                   |
| <nobr><code>{"--description"}</code></nobr> |        | Агент юу хийх ёстой вэ                                                                                                                                                                                                                      |
| <nobr><code>{"--mode"}</code></nobr>        |        | Агент горим: `all`, `primary`, эсвэл `subagent`                                                                                                                                                                                             |
| <nobr><code>{"--permissions"}</code></nobr> |        | Зөвшөөрөх зөвшөөрлийн таслалаар тусгаарлагдсан жагсаалт (өгөгдмөл: бүгд). Боломжтой: `bash`, `read`, `edit`, `glob`, `grep`, `webfetch`, `task`, `todowrite`, `websearch`, `lsp`, `skill`. Орхигдсон бүх зүйлийг үгүйсгэдэг. Нэр: `--tools` |
| <nobr><code>{"--model"}</code></nobr>       | `-m`   | Ашиглах загвар, `provider/model` формат                                                                                                                                                                                                     |

`--path`, `--description`, `--mode`, `--permissions`-г бүгдийг нь дамжуулснаар командыг интерактив бус байдлаар ажиллуулна.

---

#### жагсаалт

Боломжтой бүх агентуудыг жагсаана уу.

```bash
mongolgpt agent list
```

### хавсаргана уу

`serve` эсвэл `web` тушаалуудаар эхлүүлсэн аль хэдийн ажиллаж байгаа MongolGPT backend серверт терминалыг холбоно уу.

```bash
mongolgpt attach [url]
```

Энэ нь алсын удирдлагатай TUI-г ашиглах боломжийг олгодог. Жишээ нь:

```bash
# Вэб болон гар утаснаас хандах backend серверийг эхлүүлэх
mongolgpt web --port 4096 --hostname 0.0.0.0

# Өөр терминалаас ажиллаж буй backend-д TUI-г холбох
mongolgpt attach http://10.20.30.40:4096
```

#### Тугнууд

| Туг                                      | Богино | Тодорхойлолт                                                                                       |
| ---------------------------------------- | ------ | -------------------------------------------------------------------------------------------------- |
| <nobr><code>{"--dir"}</code></nobr>      |        | TUI-г \|-д эхлүүлэх ажлын лавлах                                                                   |
| <nobr><code>{"--continue"}</code></nobr> | `-c`   | Сүүлийн сессийг үргэлжлүүлэх                                                                       |
| <nobr><code>{"--session"}</code></nobr>  | `-s`   | Үргэлжлүүлэхийн тулд сесс ID                                                                       |
| <nobr><code>{"--fork"}</code></nobr>     |        | Үргэлжлүүлэхдээ сессийг хойшлуул (`--continue` эсвэл `--session` ашиглана уу)                      |
| <nobr><code>{"--password"}</code></nobr> | `-p`   | Баталгаажуулах үндсэн нууц үг (өгөгдмөл нь `MONGOLGPT_SERVER_PASSWORD`)                            |
| <nobr><code>{"--username"}</code></nobr> | `-u`   | Баталгаажуулах үндсэн хэрэглэгчийн нэр (өгөгдмөл нь `MONGOLGPT_SERVER_USERNAME` эсвэл `mongolgpt`) |

### баталгаажуулалт

Итгэмжлэх жуух бичгийг удирдах, үйлчилгээ үзүүлэгчдийн нэвтрэх тушаал.

```bash
mongolgpt auth [command]
```

---

#### нэвтэрнэ үү

MongolGPT нь [Models.dev](https://models.dev) хаягаар үйлчилгээ үзүүлэгчийн жагсаалтаар тэжээгддэг тул та ашиглахыг хүссэн бүх үйлчилгээ үзүүлэгчдээ API товчлуурыг тохируулахын тулд `mongolgpt auth login` ашиглаж болно. Үүнийг `~/.local/share/mongolgpt/auth.json`-д хадгална.

```bash
mongolgpt auth login
```

MongolGPT эхлэхэд итгэмжлэлийн файлаас үйлчилгээ үзүүлэгчдийг ачаална. Хэрэв таны орчинд тодорхойлогдсон түлхүүрүүд эсвэл таны төсөлд `.env` файл байгаа бол.

##### Тугнууд

| Туг                                      | Богино | Тодорхойлолт                                         |
| ---------------------------------------- | ------ | ---------------------------------------------------- |
| <nobr><code>{"--provider"}</code></nobr> | `-p`   | Үйлчилгээ үзүүлэгч ID эсвэл \| руу нэвтрэх нэр       |
| <nobr><code>{"--method"}</code></nobr>   | `-m`   | Ашиглах нэвтрэх аргын шошго, аргын сонголтыг алгасах |

---

#### жагсаалт

Итгэмжлэлийн файлд хадгалагдсан бүх баталгаажсан үйлчилгээ үзүүлэгчдийг жагсаана.

```bash
mongolgpt auth list
```

Эсвэл богино хувилбар.

```bash
mongolgpt auth ls
```

---

#### гарах

Таныг итгэмжлэлийн файлаас устгаснаар үйлчилгээ үзүүлэгчээс гарна.

```bash
mongolgpt auth logout
```

### github

Хадгалах газрын автоматжуулалтын GitHub агентийг удирдана уу.

```bash
mongolgpt github [command]
```

---

#### суулгана

GitHub агентийг өөрийн сандаа суулгана уу.

```bash
mongolgpt github install
```

Энэ нь шаардлагатай GitHub Үйлдлийн ажлын урсгалыг тохируулж, тохиргооны үйл явцыг удирдан чиглүүлдэг. [Илүү ихийг мэдэж аваарай](/docs/github).

---

#### гүйх

GitHub агентийг ажиллуул. Үүнийг ихэвчлэн GitHub Үйлдлүүдэд ашигладаг.

```bash
mongolgpt github run
```

##### Тугнууд

| Туг                                   | Тодорхойлолт                                      |
| ------------------------------------- | ------------------------------------------------- |
| <nobr><code>{"--event"}</code></nobr> | GitHub төлөө агентыг ажиллуулах хуурамч үйл явдал |
| <nobr><code>{"--token"}</code></nobr> | GitHub хувийн хандалтын токен                     |

### mcp

Загварын контекст протоколын серверүүдийг удирдах.

```bash
mongolgpt mcp [command]
```

---

#### нэмнэ үү

Өөрийн тохиргоонд MCP сервер нэмнэ үү.

```bash
mongolgpt mcp add
```

Энэ тушаал нь таныг дотоод эсвэл алсын MCP сервер нэмэхэд чиглүүлэх болно.

---

#### жагсаалт

Бүх тохируулсан MCP серверүүд болон тэдгээрийн холболтын статусыг жагсаа.

```bash
mongolgpt mcp list
```

Эсвэл богино хувилбарыг ашиглана уу.

```bash
mongolgpt mcp ls
```

---

#### баталгаажуулалт

OAuth идэвхжүүлсэн MCP серверээр баталгаажуулна уу.

```bash
mongolgpt mcp auth [name]
```

Хэрэв та серверийн нэр өгөхгүй бол OAuth боломжтой серверүүдээс сонгохыг хүсэх болно.

Та мөн OAuth чадвартай серверүүд болон тэдгээрийн баталгаажуулалтын статусыг жагсааж болно.

```bash
mongolgpt mcp auth list
```

Эсвэл богино хувилбарыг ашиглана уу.

```bash
mongolgpt mcp auth ls
```

---

#### гарах

MCP серверийн OAuth итгэмжлэлийг устгана уу.

```bash
mongolgpt mcp logout [name]
```

---

#### дибаг хийх

MCP серверийн OAuth холболтын асуудлыг дибаг хийх.

```bash
mongolgpt mcp debug <name>
```

### загвар

Тохируулсан үйлчилгээ үзүүлэгчээс боломжтой бүх загваруудыг жагсаана уу.

```bash
mongolgpt models [provider]
```

Энэ тушаал нь `provider/model` форматаар таны тохируулсан үйлчилгээ үзүүлэгчд байгаа бүх загваруудыг харуулдаг.

Энэ нь [таны тохиргоо](/docs/config/)-д яг ямар загварын нэрийг ашиглахыг олоход хэрэгтэй.

Та сонголтоороо ID үйлчилгээ үзүүлэгчээр дамжуулж загваруудыг тухайн үйлчилгээ үзүүлэгчээр шүүж болно.

```bash
mongolgpt models anthropic
```

#### Тугнууд

| Туг                                     | Тодорхойлолт                                                                |
| --------------------------------------- | --------------------------------------------------------------------------- |
| <nobr><code>{"--refresh"}</code></nobr> | models.dev \| сайтаас загварын кэшийг сэргээнэ үү                           |
| <nobr><code>{"--verbose"}</code></nobr> | Илүү дэлгэрэнгүй загвар гаралтыг ашиглах (зардал гэх мэт мета өгөгдөл орно) |

`--refresh` тугийг ашиглан кэштэй загварын жагсаалтыг шинэчилнэ үү. Энэ нь үйлчилгээ үзүүлэгчид шинэ загвар нэмэгдсэн бөгөөд та тэдгээрийг MongolGPT дээр харахыг хүсвэл хэрэг болно.

```bash
mongolgpt models --refresh
```

### гүйх

Сануулгыг шууд дамжуулж интерактив бус горимд mongolgpt-г ажиллуулна уу.

```bash
mongolgpt run [message..]
```

Энэ нь скрипт бичих, автоматжуулалт эсвэл TUI-г бүрэн эхлүүлэхгүйгээр хурдан хариулт авахыг хүссэн үед хэрэгтэй. Жишээ нь.

```bash "mongolgpt run"
mongolgpt run "Go хэлэнд context хэрхэн ашигладгийг тайлбарла"
```

Та мөн ажиллаж байгаа `mongolgpt serve` жишээнд хавсаргаж болох бөгөөд гүйлт болгонд MCP серверийг ачаалахаас зайлсхийх боломжтой.

```bash
# Нэг терминалд дэлгэцгүй сервер эхлүүлэх
mongolgpt serve

# Өөр терминалаас серверт холбогдох тушаал ажиллуулах
mongolgpt run --attach http://localhost:4096 "JavaScript-ийн async/await-ийг тайлбарла"
```

| Туг                                                          | Богино | Тодорхойлолт                                                                                       |
| ------------------------------------------------------------ | ------ | -------------------------------------------------------------------------------------------------- |
| <nobr><code>{"--command"}</code></nobr>                      |        | Ажиллуулах тушаал, args \| мессежийг ашиглана уу                                                   |
| <nobr><code>{"--continue"}</code></nobr>                     | `-c`   | Сүүлийн сессийг үргэлжлүүлэх                                                                       |
| <nobr><code>{"--session"}</code></nobr>                      | `-s`   | Үргэлжлүүлэхийн тулд сесс ID                                                                       |

| <nobr><code>{"--share"}</code></nobr>                        |        | Сессийг хуваалцах                                                                                  |
| <nobr><code>{"--model"}</code></nobr>                        | `-m`   | Үйлчилгээ үзүүлэгч/загвар хэлбэрээр ашиглах загвар                                                 |
| <nobr><code>{"--agent"}</code></nobr>                        |        | Ашиглах агент                                                                                      |
| <nobr><code>{"--file"}</code></nobr>                         | `-f`   | Мессежид хавсаргах файл(ууд)                                                                       |
| <nobr><code>{"--format"}</code></nobr>                       |        | Формат: анхдагч (форматлагдсан) эсвэл json (түүхий JSON үйл явдлууд)                               |
| <nobr><code>{"--title"}</code></nobr>                        |        | Сеансын гарчиг (утга заагаагүй бол тайрсан сануулгыг ашигладаг)                                    |
| <nobr><code>{"--attach"}</code></nobr>                       |        | Ажиллаж байгаа mongolgpt серверт хавсаргах (жишээ нь: http://localhost:4096)                       |

### үйлчилнэ

API хандахын тулд толгойгүй MongolGPT сервер ажиллуулна уу. Бүрэн HTTP интерфэйсийг [серверийн баримтууд](/docs/server)-г үзнэ үү.

```bash
mongolgpt serve
```

Энэ нь TUI интерфэйсгүйгээр mongolgpt функцэд API хандалтыг хангадаг HTTP серверийг эхлүүлнэ. HTTP үндсэн баталгаажуулалтыг идэвхжүүлэхийн тулд `MONGOLGPT_SERVER_PASSWORD`-г тохируулна уу (хэрэглэгчийн нэр нь `mongolgpt`).

#### Тугнууд

| Туг                                         | Тодорхойлолт                                       |
| ------------------------------------------- | -------------------------------------------------- |
| <nobr><code>{"--port"}</code></nobr>        | Сонсох порт                                        |
| <nobr><code>{"--hostname"}</code></nobr>    | Сонсох хостын нэр                                  |
| <nobr><code>{"--mdns"}</code></nobr>        | mDNS илрүүлэлтийг идэвхжүүлэх                      |
| <nobr><code>{"--mdns-domain"}</code></nobr> | Захиалгат mDNS домэйн нэр                          |
| <nobr><code>{"--cors"}</code></nobr>        | CORS \| зөвшөөрөх нэмэлт хөтчийн гарал үүсэл(үүд). |

| <nobr><code>{"--format"}</code></nobr>    |        | Гаралтын формат: хүснэгт эсвэл json (хүснэгт) |

### статистик

MongolGPT сессийн токен ашиглалт болон зардлын статистикийг харуул.

```bash
mongolgpt stats
```

#### Тугнууд

| Туг                                     | Тодорхойлолт                                                                                                       |
| --------------------------------------- | ------------------------------------------------------------------------------------------------------------------ |
| <nobr><code>{"--days"}</code></nobr>    | Сүүлийн N өдрийн статистикийг харуулах (бүх цаг үед)                                                               |
| <nobr><code>{"--tools"}</code></nobr>   | Харуулах хэрэгслийн тоо (бүгд)                                                                                     |
| <nobr><code>{"--models"}</code></nobr>  | Загварын ашиглалтын задаргааг харуулах (өгөгдмөлөөр нуугдсан). Шилдэг N \|-г харуулахын тулд дугаарыг дамжуулаарай |
| <nobr><code>{"--project"}</code></nobr> | Төслөөр шүүх (бүх төсөл, хоосон мөр: одоогийн төсөл)                                                               |

```bash
mongolgpt export [sessionID]
```

| ---------------------------------------- | ---------------------------------------- |

### импорт

JSON файлаас сессийн өгөгдлийг импортлох эсвэл MongolGPT URL хуваалцах.

```bash
mongolgpt import <file>
```

Та дотоод файл эсвэл MongolGPT хуваалцах URL-аас импортлох боломжтой.

```bash

```

### вэб

Вэб интерфэйстэй толгойгүй MongolGPT серверийг эхлүүл.

```bash
mongolgpt web
```

Энэ нь HTTP серверийг эхлүүлж, вэб интерфэйсээр дамжуулан MongolGPT руу хандах вэб хөтчийг нээнэ. HTTP үндсэн баталгаажуулалтыг идэвхжүүлэхийн тулд `MONGOLGPT_SERVER_PASSWORD`-г тохируулна уу (хэрэглэгчийн нэр нь `mongolgpt`).

#### Тугнууд

| Туг                                         | Тодорхойлолт                                       |
| ------------------------------------------- | -------------------------------------------------- |
| <nobr><code>{"--port"}</code></nobr>        | Сонсох порт                                        |
| <nobr><code>{"--hostname"}</code></nobr>    | Сонсох хостын нэр                                  |
| <nobr><code>{"--mdns"}</code></nobr>        | mDNS илрүүлэлтийг идэвхжүүлэх                      |
| <nobr><code>{"--mdns-domain"}</code></nobr> | Захиалгат mDNS домэйн нэр                          |
| <nobr><code>{"--cors"}</code></nobr>        | CORS \| зөвшөөрөх нэмэлт хөтчийн гарал үүсэл(үүд). |
