## Удирдагч түлхүүр

MongolGPT нь олон товчлуурын холбоосын хувьд `leader` товчлуурыг ашигладаг. Энэ нь таны терминал дахь зөрчилдөөнөөс зайлсхийх болно.

Анхдагч байдлаар, `ctrl+x` нь удирдагч түлхүүр бөгөөд олон үйлдэл хийхдээ эхлээд ахлагч товчийг дараа нь товчлолыг дарахыг шаарддаг. Жишээлбэл, шинэ сесс эхлүүлэхийн тулд эхлээд `ctrl+x`, дараа нь
` дарна уу.

Та товчлууруудаа удирдах түлхүүр ашиглах шаардлагагүй, гэхдээ бид үүнийг хийхийг зөвлөж байна.

Зарим навигацийн товчлуурууд нь анхдагчаар удирдагч товчлуурыг санаатайгаар ашигладаггүй. Дэд агент сешнүүдийн хувьд өгөгдмөл нь `session_child_first` = `<leader>down`, `session_child_cycle` = `right`, `session_child_cycle_reverse` = `left`, `session_parent` = `up`.

`leader_timeout` нь удирдагч товчлуурын дараа дараагийн түлхүүрийг хэр удаан хүлээхийг MongolGPT хянадаг. Энэ нь анхдагчаар `2000` миллисекунд байна.

## Холбох утгууд

Мөр нь нэг товчлол эсвэл таслалаар тусгаарлагдсан олон товчлол агуулж болно. Та мөн олон товчлолд массив ашиглаж болно.

Нарийвчилсан тохиолдлуудын хувьд `key`, `event`, `preventDefault` эсвэл `fallthrough` бүхий объектыг ашиглана уу.

```json title="tui.json"
{
  "$schema": "https://mgpt.mn/tui.json",
  "keybinds": {
    "messages_copy": ["<leader>y", "ctrl+shift+c"],
    "input_paste": {
      "key": "ctrl+v",
      "preventDefault": false
    }
  }
}
```

## Товчлолыг идэвхгүй болгох

Та `tui.json` дээр `"none"` эсвэл `false` гэсэн утгатай түлхүүр нэмснээр товчлуурын холбоосыг идэвхгүй болгож болно.

```json title="tui.json"
{
  "$schema": "https://mgpt.mn/tui.json",
  "keybinds": {
    "session_compact": "none"
  }
}
```

## Ширээний програмын товчлолууд

MongolGPT ширээний програмын шуурхай оролт нь текст засварлах нийтлэг Readline/Emacs маягийн товчлолуудыг дэмждэг. Эдгээр нь суурилагдсан бөгөөд одоогоор `mongolgpt.json`-ээр тохируулах боломжгүй.

| Товчлол  | Үйлдэл                                                 |
| -------- | ------------------------------------------------------ |
| `ctrl+a` | Одоогийн мөрийн эхлэл рүү шилжих                       |
| `ctrl+e` | Одоогийн мөрийн төгсгөл рүү шилжих                     |
| `ctrl+b` | Курсорыг нэг тэмдэгтээр ухраах                         |
| `ctrl+f` | Курсорыг нэг тэмдэгт урагшлуулах                       |
| `alt+b`  | Курсорыг нэг үг ухраах                                 |
| `alt+f`  | Курсорыг нэг үг урагшлуулах                            |
| `ctrl+d` | Курсорын доорх тэмдэгтийг устгах                       |
| `ctrl+k` | Мөрийн төгсгөл хүртэл алах                             |
| `ctrl+u` | Мөр эхлэх хүртэл алах                                  |
| `ctrl+w` | Өмнөх үгийг алах                                       |
| `alt+d`  | Дараагийн үгийг алах                                   |
| `ctrl+t` | Тэмдэгтүүдийг шилжүүлэх                                |
| `ctrl+g` | Поповеруудыг цуцлах / ажиллаж байгаа хариултыг зогсоох |

## Shift+Enter

Зарим терминалууд өгөгдмөлөөр Enter-тэй өөрчлөх түлхүүрүүдийг илгээдэггүй. Та `Shift+Enter`-г зугтах дараалал болгон илгээхийн тулд терминалаа тохируулах шаардлагатай байж магадгүй.

### Windows Терминал

`settings.json`-г нээнэ үү:

```
%LOCALAPPDATA%\Packages\Microsoft.WindowsTerminal_8wekyb3d8bbwe\LocalState\settings.json
```

Үүнийг root түвшний `actions` массивт нэмнэ үү:

```json
"actions": [
  {
    "command": {
      "action": "sendInput",
      "input": "\u001b[13;2u"
    },
    "id": "User.sendInput.ShiftEnterCustom"
  }
]
```

Үүнийг root түвшний `keybindings` массивт нэмнэ үү:

```json
"keybindings": [
  {
    "keys": "shift+enter",
    "id": "User.sendInput.ShiftEnterCustom"
  }
]
```

Файлыг хадгалаад Windows Терминалыг дахин эхлүүлэх эсвэл шинэ таб нээнэ үү.
