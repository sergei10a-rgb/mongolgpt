MongolGPT-ийн хуваалцах онцлог нь танд MongolGPT харилцан яриандаа олон нийтийн холбоос үүсгэх боломжийг олгодог бөгөөд ингэснээр та багийн найзуудтайгаа хамтран ажиллах эсвэл бусдаас тусламж авах боломжтой.

:::note
Хуваалцсан харилцан ярианд холбоосыг мэддэг хүн бүр нээлттэй.
:::

---

## Энэ хэрхэн ажилладаг

Та харилцан яриа хуваалцахдаа MongolGPT:

1. Таны сессийн өвөрмөц нийтийн URL үүсгэнэ
2. Таны харилцан ярианы түүхийг манай серверүүдтэй синк хийнэ
3. Харилцан яриаг хуваалцах боломжтой холбоосоор дамжуулан авах боломжтой болгоно — `opncd.ai/s/<share-id>`

---

## Хуваалцах

MongolGPT нь харилцан яриаг хэрхэн хуваалцахыг хянах гурван хуваалцах горимыг дэмждэг:

---

### Гарын авлага (өгөгдмөл)

Анхдагчаар MongolGPT гар аргаар хуваалцах горимыг ашигладаг. Сешнүүдийг автоматаар хуваалцдаггүй, гэхдээ та `/share` тушаалыг ашиглан гараар хуваалцах боломжтой.

```
/share
```

Энэ нь таны санах ойд хуулах өвөрмөц URL үүсгэх болно.

Гарын горимыг өөрийн [тохиргооны файл](/docs/config) дээр тодорхой тохируулахын тулд:

```json title="mongolgpt.json"
{
  "$schema": "https://mongolgpt.duckdns.org/config.json",
  "share": "manual"
}
```

---

### Автоматаар хуваалцах

Та өөрийн [тохиргооны файл](/docs/config) дээрх `share` сонголтыг `"auto"` гэж тохируулснаар бүх шинэ харилцан ярианд автоматаар хуваалцахыг идэвхжүүлж болно:

```json title="mongolgpt.json"
{
  "$schema": "https://mongolgpt.duckdns.org/config.json",
  "share": "auto"
}
```

Автоматаар хуваалцахыг идэвхжүүлснээр шинэ харилцан яриа бүр автоматаар хуваалцагдаж, холбоос үүсгэгдэнэ.

---

### Идэвхгүй

Та өөрийн [тохиргооны файл](/docs/config) дээрх `share` сонголтыг `"disabled"` гэж тохируулснаар хуваалцахыг бүхэлд нь идэвхгүй болгож болно:

```json title="mongolgpt.json"
{
  "$schema": "https://mongolgpt.duckdns.org/config.json",
  "share": "disabled"
}
```

Үүнийг тухайн төслийн багийнхаа хэмжээнд хэрэгжүүлэхийн тулд төслийнхөө `mongolgpt.json` дээр нэмээд Git рүү шалгана уу.

---

## Хуваалцахгүй

Харилцаа хуваалцахаа зогсоож, олон нийтийн хандалтаас хасахын тулд:

```
/unshare
```

Энэ нь хуваалцах холбоосыг устгаж, харилцан яриатай холбоотой өгөгдлийг устгах болно.

---

## Нууцлал

Ярилцлага хуваалцахдаа хэд хэдэн зүйлийг анхаарч үзэх хэрэгтэй.

---

### Өгөгдөл хадгалах

Хуваалцсан харилцан ярианууд таныг шууд хуваалцахгүй болтол хандах боломжтой хэвээр байна. Энэ
includes:

- Бүрэн харилцан ярианы түүх
- Бүх мессеж, хариулт
- Сешн мета өгөгдөл

---

### Зөвлөмж

- Зөвхөн нууц мэдээлэл агуулаагүй харилцан яриаг хуваалцах.
- Хуваалцахаасаа өмнө харилцан ярианы агуулгыг шалгана уу.
- Хамтран ажиллаж дуусмагц харилцан яриаг хуваалцахгүй.
- Өмчийн код эсвэл нууц мэдээлэлтэй харилцан яриаг хуваалцахаас зайлсхий.
- Мэдрэмжтэй төслүүдийн хувьд хуваалцахыг бүхэлд нь идэвхгүй болго.

---

## Аж ахуйн нэгжүүдэд зориулав

Байгууллагын байршуулалтын хувьд хуваалцах онцлог нь:

- **Аюулгүй байдлыг хангах үүднээс бүхэлд нь идэвхгүй болгосон**
- **Хязгаарлагдмал** зөвхөн SSO-р баталгаажуулсан хэрэглэгчдэд зориулагдсан
- **Өөртөө байршуулсан** өөрийн дэд бүтэц дээр

[Илүү ихийг мэдэж аваарай](/docs/enterprise) байгууллагадаа mongolgpt ашиглах тухай.
