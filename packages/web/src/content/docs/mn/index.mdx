[**MongolGPT**](/) бол Монгол хэрэглэгчдэд зориулсан хиймэл оюуны кодын агентын платформ. Терминал, ширээний апп, хөгжүүлэлтийн орчны өргөтгөл, үйлчилгээ үзүүлэгч, чадвар, нэмэлт болон MCP холболтыг нэг дор ажиллуулах зорилготой.

## Суулгах

Дараах баталгаатай аргуудаас өөрт тохирохыг сонгон суулгана:

- Ширээний апп: [Windows x64 суулгагч](https://github.com/sergei10a-rgb/mongolgpt/releases/latest/download/mongolgpt-desktop-win-x64.exe)
- Терминалын хувилбар: `npm install -g mongolgpt`
- Эх кодоос ажиллуулах:

```bash
git clone https://github.com/sergei10a-rgb/mongolgpt
cd mongolgpt
bun install
bun run dev
```

Терминалаас суулгах тушаал:

```bash
git clone https://github.com/sergei10a-rgb/mongolgpt && cd mongolgpt && bun install
```

## Ширээний апп суулгах

1. [Хувилбарууд](https://github.com/sergei10a-rgb/mongolgpt/releases/latest) хуудас руу орно.
2. `mongolgpt-desktop-win-x64.exe` файлыг татна.
3. Суулгагчийг ажиллуулна.

:::note
Одоогийн бүтээлтэд кодын тоон гарын үсэг хийгдээгүй тул Windows SmartScreen анхааруулга харуулж магадгүй.
:::

## Эх кодоос ажиллуулах

Кодын санг татсан бол дараах тушаалаар өөрийн төхөөрөмж дээр ажиллуулна.

```bash
bun install
bun run dev
```

Ширээний аппын хөгжүүлэлтийн горим:

```bash
bun run dev:desktop
```

Windows-ийн ширээний суулгагч бүтээх:

```powershell
$env:MONGOLGPT_CHANNEL="prod"
bun --cwd packages/desktop run package:win
```

Бүтээлт дууссаны дараа суулгагч файл энд гарна.

```txt
packages/desktop/dist/
```

## Терминал хувилбар

Терминалын хувилбар `mongolgpt` тушаалаар ажиллана. Кодын сан дахь тушаалын мөр болон нийтлэх скриптүүд энэ нэрийг ашиглана.

Терминалын хувилбарыг npm-ээс суулгана:

```bash
npm install -g mongolgpt
mongolgpt
```

Харин хөгжүүлэлт хийх бол эх кодоос ажиллуулна:

```bash
git clone https://github.com/sergei10a-rgb/mongolgpt
cd mongolgpt
bun install
bun run dev
```

Мөн дараах суулгах тушаалыг ашиглаж болно:

```bash
git clone https://github.com/sergei10a-rgb/mongolgpt && cd mongolgpt && bun install
```

## Нэвтрэх ба үйлчилгээ үзүүлэгч тохируулах

:::caution[Одоогийн төлөв]
Нэгдсэн нэвтрэлт, Free Auto, quota болон төлбөрийн backend dev орчинд туршигдаж байна. Production нээлтийг release notes-д зарлахаас өмнө бодит төлбөрийн мэдээлэл бүү оруул.
:::

Hosted service идэвхтэй release дээр MongolGPT Desktop-ийг анх нээхэд, мөн `mongolgpt` тушаалыг анх ажиллуулахад MongolGPT бүртгэлээр нэвтрэх цонх нээгдэнэ. Вэб хөтөч дээр бүртгүүлэх эсвэл нэвтэрч зөвшөөрсний дараа апп руу автоматаар буцна.

Нэвтэрсний дараа **MongolGPT Free Auto** анхдагчаар бэлэн болно. Free Auto нь тухайн үед боломжтой загвар руу автоматаар чиглүүлж, таны бүртгэлийн хязгаарыг мөрдөнө. Үйлчилгээний боломж, загвар, хязгаар өөрчлөгдөж болох тул үүрд хязгааргүй, үнэгүй байна гэсэн амлалт биш.

Нэмэлт үйлчилгээ үзүүлэгч холбохдоо `/connect` ашиглана.

```txt
/connect
```

1. OpenAI, Anthropic, Google, OpenRouter зэрэг үйлчилгээг өөрийн API түлхүүр эсвэл OAuth эрхээр холбож болно.
2. [Ollama](./providers#оллама), [LM Studio](./providers#lm-studio) болон [OpenAI-тэй нийцтэй тусгай үйлчилгээ](./providers#тусгай-үйлчилгээ-үзүүлэгч) ашиглан орон нутгийн загвар холбоно.
3. NVIDIA NIM-ийг өөрийн API түлхүүрээр сонголтоор холбоно. NVIDIA-ийн байршуулсан туршилтын API эрх нь хувийн хөгжүүлэлт, туршилт, үнэлгээнд зориулагддаг; үйлдвэрлэлийн хэрэглээнд зохих лиценз эсвэл захиалга шаардлагатай.

MongolGPT бүртгэл нь Free Auto болон MongolGPT-ийн удирддаг үйлчилгээнд шаардлагатай. Өөрийн API түлхүүр, локал загварын нууц мэдээлэл хэрэглэгчийн төхөөрөмж дээрх MongolGPT нууц мэдээллийн үйлчилгээнд хадгалагдана. Дэлгэрэнгүйг [бүртгэл, Free Auto ба багцын заавраас](/docs/account) үзнэ үү.

## Чадвар, нэмэлт, MCP

MongolGPT-ийн зорилго нь Claude, Codex, Goose, MCP сервер, чадвар болон нэмэлт зэрэг өөр өөр агентын форматыг таньж, шаардлагатай тааруулагчийг хэрэглэгчээр гараар бичүүлэхгүйгээр үүсгэн холбох юм.

Эхний суурь нь repo дотор байна:

- үйлчилгээ үзүүлэгчийн холболт
- нэмэлтийн API-ийн суурь
- MCP холболтын суурь
- ширээний апп
- терминалд ажиллах агент
- MongolGPT бүртгэлийн нэвтрэлтийн холболт

Нийцтэй байдлын импортлогчийн эхний ажиллах хувилбар нэмэгдсэн:

- тушаал, дотоод зам, URL, тохиргооны файлыг уншиж MCP, чадвар, нэмэлтийн төрлийг танина
- `plan` горимоор тохиргоонд орох өөрчлөлтийг эхлээд харуулна
- `apply` горимоор нийтлэг эсвэл төслийн тохиргоонд аюулгүйгээр нэмнэ
- нэмэлтийн тааруулагч шаардлагатай бол хэрэглэгчээр гараар бичүүлэхгүйгээр үүсгэх суурьтай

Терминал дээр:

```bash
mongolgpt compat import plan "npx -y @modelcontextprotocol/server-filesystem C:\\Users\\me"
mongolgpt compat import apply "npx -y @modelcontextprotocol/server-filesystem C:\\Users\\me"
```

Ширээний аппын `Тохиргоо -> Интеграц` хэсэгт тушаал, дотоод зам, URL эсвэл тохиргоо оруулж шалгаад суулгаж болно.

Дараагийн шатанд нэмэлтийн сангаас хайх, илүү олон агентын форматад тааруулагч үүсгэх, гарын үсэг болон зөвшөөрлийг шалгах боломж нэмэгдэнэ.
