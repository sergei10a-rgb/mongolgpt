[**MongolGPT**](/) бол Монгол хэрэглэгчдэд зориулсан AI coding agent платформ. Терминал, desktop app, IDE өргөтгөл, provider, skill/plugin, MCP холболтуудыг нэг дор ажиллуулах зорилготой.

## Суулгах

Одоогоор MongolGPT-ийн албан ёсны npm/Homebrew/AUR/Chocolatey/Scoop package нийтлэгдээгүй. Бодитоор ажиллах замууд:

- Desktop: [Windows x64 installer](https://github.com/sergei10a-rgb/mongolgpt/releases/latest/download/mongolgpt-desktop-win-x64.exe)
- Source build:

```bash
git clone https://github.com/sergei10a-rgb/mongolgpt
cd mongolgpt
bun install
bun run dev
```

Terminal install script:

```bash
git clone https://github.com/sergei10a-rgb/mongolgpt && cd mongolgpt && bun install
```

## Desktop суулгах

1. [Releases](https://github.com/sergei10a-rgb/mongolgpt/releases/latest) хуудас руу орно.
2. `mongolgpt-desktop-win-x64.exe` файлыг татна.
3. Суулгагчийг ажиллуулна.

:::note
Одоогийн build code signing хийгдээгүй. Windows SmartScreen анхааруулга харуулж магадгүй.
:::

## Source-оос ажиллуулах

Repository-г clone хийсэн бол дараах байдлаар локал орчинд ажиллуулна.

```bash
bun install
bun run dev
```

Desktop development mode:

```bash
bun run dev:desktop
```

Windows desktop installer build хийх:

```powershell
$env:MONGOLGPT_CHANNEL="prod"
bun --cwd packages/desktop run package:win
```

Build дууссаны дараа installer файл энд гарна.

```txt
packages/desktop/dist/
```

## Терминал хувилбар

Terminal CLI нь `mongolgpt` command нэрээр ажиллах ёстой. Repository дотор CLI болон publish script-үүд энэ нэр рүү шилжсэн.

Registry package нийтлэгдэх хүртэл terminal хувилбарыг source-оос ажиллуулна:

```bash
git clone https://github.com/sergei10a-rgb/mongolgpt
cd mongolgpt
bun install
bun run dev
```

Мөн install script туршиж болно:

```bash
git clone https://github.com/sergei10a-rgb/mongolgpt && cd mongolgpt && bun install
```

## Нэвтрэх ба provider тохируулах

MongolGPT account-аар нэвтрэх урсгал нь `https://mongolgpt.duckdns.org/auth` endpoint руу чиглэнэ.

```txt
/connect
```

`/connect` командаар provider сонгоод API key эсвэл MongolGPT account холболтоо тохируулна.

## Skill, plugin, MCP

MongolGPT-ийн зорилго нь Claude, Codex, Goose, MCP server, skill/plugin зэрэг өөр өөр AI agent format-уудыг хэрэглэгчээр гараар тааруулахгүйгээр таньж, шаардлагатай үед adapter/wrapper үүсгэн холбох юм.

Эхний суурь нь repo дотор байна:

- provider integration
- plugin API суурь
- MCP холболтын суурь
- desktop shell
- terminal agent
- MongolGPT account login wiring

Шинэ compatibility importer нь эхний ажиллах хэлбэрээр нэмэгдсэн:

- command, локал зам, URL, config файлыг уншиж MCP, skill, plugin төрлийг танина
- `plan` горимоор ямар config patch үүсэхийг эхлээд харуулна
- `apply` горимоор global эсвэл project config-д аюулгүйгээр нэмнэ
- plugin wrapper шаардлагатай бол хэрэглэгчээр гараар бичүүлэхгүйгээр үүсгэх суурьтай

CLI дээр:

```bash
mongolgpt compat import plan "npx -y @modelcontextprotocol/server-filesystem C:\\Users\\me"
mongolgpt compat import apply "npx -y @modelcontextprotocol/server-filesystem C:\\Users\\me"
```

Desktop дээр `Тохиргоо -> Интеграц` хэсгээс command, path, URL эсвэл config оруулаад шалгах болон суулгах боломжтой.

Дараагийн шатанд marketplace discovery, илүү олон agent format-ийн adapter generator, signing/permission review нэмэгдэнэ.
