MongolGPT-ийн удирддаг загваруудыг ширээний апп, вэб болон терминалаас нэг бүртгэлээр ашиглана. Үүнийг ажиллуулахын тулд MongolGPT-ийн нэвтрэлт, удирдлагын самбар, загварын гарц болон үйлдвэрлэлийн үйлчилгээ үзүүлэгчдийг байршуулсан байх шаардлагатай.

:::caution
Үйлдвэрлэлийн байршуулалт дуусаж, албан ёсны нөхцөл нийтлэгдэхээс өмнө загварын нэр, үнэ, хэрэглээний хязгаар болон бэлэн байдлыг баталгаатай гэж үзэж болохгүй.
:::

## Нэвтрэх

MongolGPT-ийг анх ажиллуулахад вэб хөтөч дээр бүртгүүлэх эсвэл нэвтэрнэ. OAuth хандалтын болон шинэчлэх токеныг эх кодын тохиргоонд бичихгүй, хэрэглэгчийн төхөөрөмжийн нууц мэдээллийн санд хадгална.

Үйлдвэрлэлийн үйлчилгээ идэвхжсэний дараа нэвтэрсэн хэрэглэгчид **MongolGPT Free Auto** анхдагч сонголт болно. Бүртгэлийн токенгүй хүсэлтийг загварын гарц хүлээн авахгүй.

## Free Auto

Free Auto нь тухайн үед боломжтой үйлдвэрлэлийн загвар руу автоматаар чиглүүлнэ. Үндсэн үйлчилгээ үзүүлэгч ажиллахгүй бол зөвхөн урьдчилан тохируулсан нөөц үйлчилгээ үзүүлэгч рүү шилжинэ.

- зөвхөн нэвтэрсэн MongolGPT бүртгэлд ажиллана;
- минутын хүсэлтийн болон долоо хоногийн токены хязгаартай;
- NVIDIA-ийн туршилтын API-г нийтийн backend болгон ашиглахгүй;
- боломжтой загвар, хэрэглээний хязгаар болон үйлчилгээний нөхцөл өөрчлөгдөж болно;
- үүрд хязгааргүй, үнэгүй байна гэсэн амлалт биш.

Хязгаарт хүрвэл MongolGPT дахин ашиглаж болох хугацааг Монгол хэлээр харуулна.

## Нэмэлт үйлчилгээ үзүүлэгч

Free Auto-оос гадна `/connect` тушаалаар өөрийн үйлчилгээ үзүүлэгчийг сонголтоор холбоно.

```txt
/connect
```

Дэмжигдэх сонголтод OpenAI, Anthropic, Google, OpenRouter, өөрийн API түлхүүртэй NVIDIA NIM, Ollama, LM Studio болон OpenAI-тэй нийцтэй дотоод үйлчилгээ орно. NVIDIA-ийн байршуулсан туршилтын API эрх нь хувийн хөгжүүлэлт, туршилт, үнэлгээнд зориулагддаг; үйлдвэрлэлийн хэрэглээнд зохих лиценз эсвэл захиалга шаардлагатай.

## API

Үйлдвэрлэлийн байршуулалтын дараа OpenAI-тэй нийцтэй үндсэн зам:

```txt
https://mgpt.mn/zen/v1
```

Боломжтой загвар болон яг ашиглах загварын ID-г динамик жагсаалтаас авна:

```txt
GET https://mgpt.mn/zen/v1/models
```

Загварын ID, үйлчилгээ үзүүлэгчийн ID, багцын нэр болон API замыг орчуулж өөрчлөхгүй. Жишээ нь `free-auto` ID-г тохиргоонд `mongolgpt/free-auto` хэлбэрээр ашиглана.

## Үнэ ба хязгаар

Үнэ, Basic/Pro/Max багц, Free Auto-ийн хэрэглээний хязгаар болон төлбөрийн нөхцөлийг нээлтийн өмнө эцэслэн батална. Код дотор хурдан хуучрах тогтмол үнийн хүснэгт хадгалахгүй; хүчинтэй мэдээллийг MongolGPT бүртгэлийн самбар болон албан ёсны төлбөрийн нөхцөлд харуулна.

## Нууцлал

Загварын гарц нь хүсэлт, токены хэрэглээ, үйлчилгээ үзүүлэгч, ажлын талбар болон тооцоонд шаардлагатай мета өгөгдлийг хэмжинэ. Даалгаврын агуулга, эх код, загварын хариуг энэ гарцын хэмжилтийн бүртгэлд бичихгүй. Гуравдагч үйлчилгээ үзүүлэгч ашиглах үед тухайн үйлчилгээний өгөгдөл хадгалах болон сургалтын бодлого давхар үйлчилнэ.

Нууц болон хувийн мэдээлэл илгээхийн өмнө сонгосон үйлчилгээ үзүүлэгчийн нөхцөлийг шалгана уу. Асуух зүйл байвал <a href={email}>MongolGPT багтай холбогдоно уу</a>.
