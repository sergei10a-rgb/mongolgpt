## GitLab CI

MongolGPT ердийн GitLab шугам хоолойд ажилладаг. Та үүнийг [CI бүрэлдэхүүн хэсэг](https://docs.gitlab.com/ee/ci/components/) хэлбэрээр дамжуулах хоолой болгон барьж болно.

Энд бид олон нийтийн үүсгэсэн CI/CD бүрэлдэхүүнийг MongolGPT — [nagyv/gitlab-mongolgpt](https://gitlab.com/nagyv/gitlab-mongolgpt) ашиглаж байна.

---

### Онцлогууд

- **Ажил бүрт тусгай тохиргоог ашиглах**: MongolGPT-г захиалгат тохиргооны лавлахаар тохируулна уу, жишээ нь MongolGPT дуудлагын функцийг идэвхжүүлэх эсвэл идэвхгүй болгохын тулд `./config/#custom-directory`.
- **Хамгийн бага тохиргоо**: CI бүрэлдэхүүн нь цаана нь MongolGPT-г тохируулдаг тул та зөвхөн MongolGPT тохиргоо болон анхны сануулгыг үүсгэхэд л хангалттай.
- **Уян хатан**: CI бүрэлдэхүүн хэсэг нь зан төлөвийг тохируулах хэд хэдэн оролтыг дэмждэг.

---

### Тохиргоо

1. MongolGPT нэвтрэлт танилтаа JSON файлын төрлийн CI орчны хувьсагчийг **Тохиргоо** > **CI/CD** > **Хувьсагч** гэсэн хэсэгт хадгална уу. Тэдгээрийг "Масклагдсан ба далд" гэж тэмдэглэхээ мартуузай.
2. `.gitlab-ci.yml` файлдаа дараахыг нэмнэ үү.

   ```yaml title=".gitlab-ci.yml"
   include:
     - component: $CI_SERVER_FQDN/nagyv/gitlab-mongolgpt/mongolgpt@2
       inputs:
         config_dir: ${CI_PROJECT_DIR}/mongolgpt-config
         auth_json: $MONGOLGPT_AUTH_JSON # MongolGPT нэвтрэлтийн JSON хадгалсан хувьсагч
         command: optional-custom-command
         message: "Таны сануулга энд"
   ```

Илүү их мэдээлэл авах бол энэ бүрэлдэхүүн хэсгийн хувьд [баримтуудыг шалгана уу](https://gitlab.com/explore/catalog/nagyv/gitlab-mongolgpt) ашиглана уу.

## GitLab Duo

MongolGPT нь таны GitLab ажлын урсгалтай нэгддэг.
Сэтгэгдэл бичихдээ `@mongolgpt`-г дурдвал MongolGPT таны GitLab CI шугаман дээрх ажлуудыг гүйцэтгэх болно.

---

### Онцлогууд

- **Гурвалсан асуудлууд**: MongolGPT-аас асуудлыг судалж, танд тайлбарлахыг хүс.
- **Засаж хэрэгжүүлэх**: Асуудлыг засах эсвэл функцийг хэрэгжүүлэхийг MongolGPT-аас хүс.
  Энэ нь шинэ салбар үүсгэж, өөрчлөлттэй хамт нэгтгэх хүсэлт гаргах болно.
- **Аюулгүй**: MongolGPT таны GitLab runner дээр ажиллана.

---

### Тохиргоо

MongolGPT нь таны GitLab CI/CD шугаманд ажилладаг тул үүнийг тохируулахын тулд танд дараах зүйлс хэрэгтэй болно:

:::tip
Хамгийн сүүлийн үеийн зааварчилгааг [**GitLab баримтууд**](https://docs.gitlab.com/user/duo_agent_platform/agent_assistant/)-г үзнэ үү.
:::

1.  Өөрийн GitLab орчныг тохируулна уу
2.  CI/CD-г тохируулах
3.  AI загвар нийлүүлэгч API түлхүүр аваарай
4.  Үйлчилгээний данс үүсгэх
5.  CI/CD хувьсагчдыг тохируулах
6.  Урсгалын тохиргооны файл үүсгэ, жишээ нь:

        <details>

    <summary>Урсгалын тохиргоо</summary>

    ```yaml
    image: node:22-slim
    commands:
      - echo "MongolGPT CLI суулгаж байна"
      - npm install --global mongolgpt@0.1.1
      - echo "glab суулгаж байна"
      - export GITLAB_TOKEN=$GITLAB_TOKEN_MONGOLGPT
      - apt-get update --quiet && apt-get install --yes curl wget gpg git && rm --recursive --force /var/lib/apt/lists/*
      - curl --silent --show-error --location "https://raw.githubusercontent.com/upciti/wakemeops/main/assets/install_repository" | bash
      - apt-get install --yes glab
      - echo "glab тохируулж байна"
      - echo $GITLAB_HOST
      - echo "MongolGPT нэвтрэлтийн тохиргоо үүсгэж байна"
      - mkdir --parents ~/.local/share/mongolgpt
      - |
        cat > ~/.local/share/mongolgpt/auth.json << EOF
        {
          "anthropic": {
            "type": "api",
            "key": "$ANTHROPIC_API_KEY"
          }
        }
        EOF
      - echo "git тохируулж байна"
      - git config --global user.email "mongolgpt@gitlab.com"
      - git config --global user.name "MongolGPT"
      - echo "glab-ийг шалгаж байна"
      - glab issue list
      - echo "MongolGPT ажиллуулж байна"
      - |
        mongolgpt run "
        Та GitLab ажиллагаанд туслах хиймэл оюуны агент.

        Контекст: $AI_FLOW_CONTEXT
        Даалгавар: $AI_FLOW_INPUT
        Үйл явдал: $AI_FLOW_EVENT

        Боломжтой GitLab хэрэгслээр хүссэн даалгаврыг гүйцэтгэ.
        Нягт шинжилж, ойлгомжтой тайлбар өг.

        <important>
        GitLab өгөгдөлд хандахдаа glab CLI ашигла. glab CLI аль хэдийн нэвтэрсэн тул шаардлагатай тушаалыг ажиллуулж болно.

        MR эсвэл issue-г дүгнэх, нэмэлт мэдээлэл өгөх хүсэлт ирвэл хэрэглэгч харах боломжтойгоор тухайн MR/issue дээр тэмдэглэл үлдээ.
        Өөрчлөлтийг commit эсвэл push хийх шаардлагагүй. Файлын өөрчлөлтөөс хамааран автоматаар хийгдэнэ.
        </important>
        "
      - git checkout --branch $CI_WORKLOAD_REF origin/$CI_WORKLOAD_REF
      - echo "git өөрчлөлт шалгаж байна"
      - |
        if ! git diff --quiet || ! git diff --cached --quiet || [ --not --zero "$(git ls-files --others --exclude-standard)" ]; then
          echo "git өөрчлөлт илэрлээ, нэмээд push хийж байна..."
          git add .
          if git diff --cached --quiet; then
            echo "Commit хийх staged өөрчлөлт алга"
          else
            echo "$CI_WORKLOAD_REF салбарт өөрчлөлт commit хийж байна"
            git commit --message "MongolGPT өөрчлөлтүүд"
            echo "$CI_WORKLOAD_REF рүү push хийж байна"
            git push https://gitlab-ci-token:$GITLAB_TOKEN@$GITLAB_HOST/gl-demo-ultimate-dev-ai-epic-17570/test-java-project.git $CI_WORKLOAD_REF
            echo "Өөрчлөлтийг амжилттай push хийлээ"
          fi
        else
          echo "git өөрчлөлт алга, push хийхгүй"
        fi
    variables:
      - ANTHROPIC_API_KEY
      - GITLAB_TOKEN_MONGOLGPT
      - GITLAB_HOST
    ```

        </details>

Та нарийвчилсан зааврыг [GitLab CLI агентуудын баримт бичиг](https://docs.gitlab.com/user/duo_agent_platform/agent_assistant/) хаягаар лавлана уу.

### Жишээ

GitLab-д MongolGPT-г хэрхэн ашиглаж болох зарим жишээ энд байна.

:::tip
Та `@mongolgpt`-с өөр гох хэллэг ашиглахаар тохируулж болно.
:::

- **Асуудлыг тайлбарлана уу**

  Энэ сэтгэгдлийг GitLab дугаарт нэмнэ үү.

  ```
  @mongolgpt энэ issue-г тайлбарла
  ```

  MongolGPT асуудлыг уншиж, тодорхой тайлбартайгаар хариулах болно.

- **Асуудлыг засах**

  GitLab дугаарт:

  ```
  @mongolgpt үүнийг зас
  ```

  MongolGPT нь шинэ салбар үүсгэж, өөрчлөлтийг хэрэгжүүлж, өөрчлөлттэй нэгтгэх хүсэлтийг нээнэ.

- **Нэгдэх хүсэлтийг хянан үзэх**

  GitLab нэгтгэх хүсэлтийн талаар дараах тайлбарыг үлдээнэ үү.

  ```
  @mongolgpt энэ merge request-ийг хяна
  ```

  MongolGPT нэгтгэх хүсэлтийг хянаж, санал хүсэлтээ өгнө.
