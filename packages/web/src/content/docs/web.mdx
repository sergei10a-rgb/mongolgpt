MongolGPT нь таны хөтөч дээр вэб программ хэлбэрээр ажиллах боломжтой бөгөөд энэ нь терминал шаардлагагүйгээр ижил хүчирхэг AI кодлох туршлагыг хангана.

![MongolGPT Web - Шинэ сесс](../../assets/web/web-homepage-new-session.png)

## Эхлэл

Вэб интерфэйсийг эхлүүлэхийн тулд:

```bash
mongolgpt web
```

Энэ нь `127.0.0.1` дээрх локал серверийг санамсаргүй боломжтой портоор эхлүүлж, таны анхдагч хөтөч дээр MongolGPT автоматаар нээгдэнэ.

:::caution
Хэрэв `MONGOLGPT_SERVER_PASSWORD`-г тохируулаагүй бол сервер хамгаалалтгүй болно. Энэ нь орон нутгийн хэрэглээнд тохиромжтой боловч сүлжээнд хандахад тохируулсан байх ёстой.
:::

:::tip[Windows Хэрэглэгчид]
Хамгийн сайн туршлага авахын тулд PowerShell-ийн оронд `mongolgpt web`-ийг [WSL](/docs/windows-wsl)-ээс ажиллуулна уу. Энэ нь файлын системийн зөв хандалт болон терминалын нэгтгэлийг баталгаажуулдаг.
:::

## Тохиргоо

Та вэб серверийг командын мөрийн туг ашиглан эсвэл [тохиргооны файл](/docs/config)-д тохируулж болно.

### Порт

Анхдагчаар MongolGPT боломжтой портыг сонгоно. Та портыг зааж өгч болно:

```bash
mongolgpt web --port 4096
```

### Хостын нэр

Анхдагчаар сервер нь `127.0.0.1` (зөвхөн локал хост) руу холбогддог. MongolGPT-г сүлжээндээ хандах боломжтой болгохын тулд:

```bash
mongolgpt web --hostname 0.0.0.0
```

`0.0.0.0`-г ашиглах үед MongolGPT дотоод болон сүлжээний хаягийг харуулна:

```
  Local access:       http://localhost:4096
  Network access:     http://192.168.1.100:4096
```

### mDNS Discovery

Дотоод сүлжээнд өөрийн серверийг харуулахын тулд mDNS-г идэвхжүүлнэ үү:

```bash
mongolgpt web --mdns
```

Энэ нь автоматаар хостын нэрийг `0.0.0.0` болгож серверийг `mongolgpt.local` гэж сурталчилдаг.

Та нэг сүлжээнд олон тохиолдлыг ажиллуулахын тулд mDNS домэйн нэрийг өөрчилж болно:

```bash
mongolgpt web --mdns --mdns-domain myproject.local
```

### CORS

CORS-д нэмэлт домэйныг зөвшөөрөхийн тулд (захиалгат нүүрэнд хэрэгтэй):

```bash
mongolgpt web --cors https://example.com
```

### Баталгаажуулалт

Хандалтыг хамгаалахын тулд `MONGOLGPT_SERVER_PASSWORD` орчны хувьсагчийг ашиглан нууц үг тохируулна уу:

```bash
MONGOLGPT_SERVER_PASSWORD=secret mongolgpt web
```

Хэрэглэгчийн нэр нь анхдагчаар `mongolgpt` боловч `MONGOLGPT_SERVER_USERNAME` гэж өөрчлөх боломжтой.

## Вэб интерфэйс ашиглах

Ажиллаж эхэлмэгц вэб интерфэйс нь таны MongolGPT сессүүдэд хандах боломжийг олгоно.

### Сеанс

Нүүр хуудаснаас өөрийн сессүүдийг харж удирдах боломжтой. Та идэвхтэй сессүүдийг харж, шинээр эхлүүлэх боломжтой.

![MongolGPT Web - Идэвхтэй сесс](../../assets/web/web-homepage-active-session.png)

### Серверийн төлөв

Холбогдсон серверүүд болон тэдгээрийн статусыг харахын тулд "Серверүүдийг харах" дээр дарна уу.

![MongolGPT Web - Серверүүдийг үзнэ үү](../../assets/web/web-homepage-see-servers.png)

## Терминал холбох

Та ажиллаж байгаа вэб серверт TUI терминал хавсаргаж болно:

```bash
# Start the web server
mongolgpt web --port 4096

# In another terminal, attach the TUI
mongolgpt attach http://localhost:4096
```

Энэ нь танд вэб интерфэйс болон терминалыг нэгэн зэрэг ашиглах, ижил сесс болон төлөвийг хуваалцах боломжийг олгоно.

## Тохиргооны файл

Та мөн `mongolgpt.json` тохиргооны файлдаа серверийн тохиргоог хийж болно:

```json
{
  "server": {
    "port": 4096,
    "hostname": "0.0.0.0",
    "mdns": true,
    "cors": ["https://example.com"]
  }
}
```

Командын мөрийн туг нь тохиргооны файлын тохиргооноос давуу эрхтэй.
