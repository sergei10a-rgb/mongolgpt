MongolGPT-ийн тусламжтайгаар та хэд хэдэн суулгасан загвараас аль нэгийг нь сонгох, терминалын загварт тохирсон загвар ашиглах эсвэл өөрийн хүссэн загвараа тодорхойлох боломжтой.

Анхдагч байдлаар MongolGPT нь бидний өөрийн `mongolgpt` загварыг ашигладаг.

---

## Терминалын шаардлага

Загваруудыг өнгөт палитраараа зөв харуулахын тулд таны терминал **truecolor** (24 битийн өнгө)-г дэмжих ёстой. Ихэнх орчин үеийн терминалууд үүнийг анхдагчаар дэмждэг боловч та үүнийг идэвхжүүлэх шаардлагатай байж магадгүй:

- **Дэмжлэгийг шалгах**: `echo $COLORTERM` ажиллуулах - энэ нь `truecolor` эсвэл `24bit` гаргана.
- ** Truecolor-г идэвхжүүлэх**: Өөрийн бүрхүүлийн профайлд `COLORTERM=truecolor` орчны хувьсагчийг тохируулна уу
- **Терминалын нийцтэй байдал**: Таны терминал эмулятор 24 битийн өнгийг дэмждэг эсэхийг шалгаарай (iTerm2, Alacritty, Kitty, Windows Терминал гэх мэт хамгийн орчин үеийн терминалууд болон GNOME Терминалын сүүлийн хувилбарууд)

Truecolor-ийн дэмжлэггүйгээр загварууд өнгөний нарийвчлал багатай гарч ирэх эсвэл хамгийн ойрын 256 өнгө хүртэл буурч болно.

## Системийн сэдэв

`system` загвар нь таны терминалын өнгөний схемд автоматаар дасан зохицоход зориулагдсан. Тогтмол өнгө ашигладаг уламжлалт загваруудаас ялгаатай нь _системийн_ сэдэв:

- **Саарал хуваарийг үүсгэдэг**: Таны терминалын дэвсгэр өнгө дээр үндэслэн тохируулсан саарал масштабыг үүсгэж, оновчтой тодосгогчийг хангана.
- ** ANSI өнгийг ашигладаг**: Таны терминалын өнгөт палитрыг хүндэтгэдэг синтаксийг тодруулах, UI элементүүдэд стандарт ANSI өнгийг (0-15) ашигладаг.
- **Терминалын өгөгдмөл тохиргоог хадгалдаг**: Терминалынхаа эх төрхийг хадгалахын тулд текст болон дэвсгэр өнгөнд
one`-г ашигладаг.

Системийн сэдэв нь дараах хэрэглэгчдэд зориулагдсан:

- MongolGPT-г терминалынхаа дүр төрхтэй тааруулахыг хүсч байна
- Захиалгат терминалын өнгөний схемийг ашиглах
- Бүх терминалын аппликешн дээр тууштай харагдахыг илүүд үздэг

---

## Загвар ашиглаж байна

Та `/theme` командыг ашиглан сэдвийг сонгох боломжтой. Эсвэл та үүнийг `tui.json`-д зааж өгч болно.

```json title="tui.json" {3}
{
  "$schema": "https://mongolgpt.duckdns.org/tui.json",
  "theme": "tokyonight"
}
```

---

## Захиалгат загварууд

MongolGPT нь хэрэглэгчдэд хялбархан загвар үүсгэж, өөрчлөх боломжийг олгодог JSON- дээр суурилсан уян хатан загварын системийг дэмждэг.

---

### Шатлал

Сэдвүүдийг хэд хэдэн сангаас дараах дарааллаар ачаалдаг бол хожмын сангууд өмнөх сангуудыг дардаг.

1. **Баригдсан загварууд** - Эдгээр нь хоёртын файлд суулгагдсан
2. **Хэрэглэгчийн тохиргооны лавлах** - `~/.config/mongolgpt/themes/*.json` эсвэл `$XDG_CONFIG_HOME/mongolgpt/themes/*.json`-д тодорхойлогдсон.
3. **Төслийн үндсэн лавлах** - `<project-root>/.mongolgpt/themes/*.json`-д тодорхойлогдсон.
4. **Одоогийн ажлын лавлах** - `./.mongolgpt/themes/*.json`-д тодорхойлогдсон

Хэрэв олон лавлах нь ижил нэртэй сэдэв агуулж байвал илүү давуу эрхтэй лавлахын сэдвийг ашиглана.

---

### Загвар үүсгэх

Захиалгат загвар үүсгэхийн тулд сэдвийн лавлахуудын аль нэгэнд JSON файл үүсгэнэ үү.

Хэрэглэгчийн өргөн хүрээний сэдвүүдийн хувьд:

```bash no-frame
mkdir -p ~/.config/mongolgpt/themes
vim ~/.config/mongolgpt/themes/my-theme.json
```

Мөн төслийн тусгай сэдвүүдийн хувьд.

```bash no-frame
mkdir -p .mongolgpt/themes
vim .mongolgpt/themes/my-theme.json
```

---

### JSON формат

Загварууд нь уян хатан JSON форматыг ашигладаг:

- **Hex өнгө**: `"#ffffff"`
- **ANSI өнгө**: `3` (0-255)
- **Өнгөний лавлагаа**: `"primary"` эсвэл захиалгат тодорхойлолт
- **Харанхуй/цайвар хувилбарууд**: `{"dark": "#000", "light": "#fff"}`
- **Өнгөгүй**: `"none"` - Терминалын өгөгдмөл өнгө буюу тунгалаг өнгийг ашиглана

---

### Өнгөний тодорхойлолт

`defs` хэсэг нь сонголттой бөгөөд энэ нь сэдэвт лавлаж болох дахин ашиглах боломжтой өнгийг тодорхойлох боломжийг танд олгоно.

---

### Терминалын өгөгдмөл

`"none"` тусгай утгыг терминалын анхдагч өнгийг өвлөхийн тулд ямар ч өнгөт ашиглаж болно. Энэ нь таны терминалын өнгөний схемд нийцсэн сэдвүүдийг бий болгоход онцгой ач холбогдолтой:

- `"text": "none"` - Терминалын өгөгдмөл нүүрэн талын өнгийг ашигладаг
- `"background": "none"` - Терминалын үндсэн дэвсгэр өнгийг ашигладаг
