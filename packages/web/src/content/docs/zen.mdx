MongolGPT Zen бол MongolGPT багийн өгсөн туршиж, баталгаажуулсан загваруудын жагсаалт юм.

Zen нь MongolGPT дахь бусад үйлчилгээ үзүүлэгчтэй адил ажилладаг. Та MongolGPT Zen руу нэвтэрч, аваарай
таны API түлхүүр. Энэ нь **бүрэн сонголттой** бөгөөд ашиглахын тулд үүнийг ашиглах шаардлагагүй
MongolGPT.

## Дэвсгэр

Маш олон тооны загварууд байдаг, гэхдээ цөөхөн нь л байдаг
Эдгээр загварууд нь кодлогчийн хувьд сайн ажилладаг. Үүнээс гадна ихэнх үйлчилгээ үзүүлэгчид байдаг
маш өөрөөр тохируулагдсан; Ингэснээр та маш өөр гүйцэтгэл, чанарыг олж авах болно.

:::tip
Бид MongolGPT-тэй сайн ажилладаг загвар, үйлчилгээ үзүүлэгчдийн сонгогдсон бүлгийг туршиж үзсэн.
:::

Хэрэв та OpenRouter шиг загвар ашиглаж байгаа бол хэзээ ч байж чадахгүй
Та хүссэн загварынхаа хамгийн сайн хувилбарыг авч байгаа эсэхээ шалгаарай.

Үүнийг засахын тулд бид хэд хэдэн зүйлийг хийсэн:

1. Бид сонгосон загвар өмсөгчдийг туршиж үзээд, хэрхэн хийх талаар багуудтай нь ярилцсан
   тэднийг ажиллуулах нь дээр.
2. Дараа нь бид эдгээр үйлчилгээ үзүүлж байгаа эсэхийг шалгахын тулд цөөн хэдэн үйлчилгээ үзүүлэгчтэй ажилласан
   зөв.
3. Эцэст нь бид загвар/үйлчилгээ үзүүлэгчийн хослолыг харьцуулж үзээд гарч ирлээ
   Бидний санал болгож буй жагсаалтын хамт.

MongolGPT Zen нь эдгээр загварт нэвтрэх боломжийг олгодог AI гарц юм.

---

## Энэ хэрхэн ажилладаг

MongolGPT Zen нь MongolGPT дахь бусад үйлчилгээ үзүүлэгчтэй адил ажилладаг.

1. Та **<a href={console}>MongolGPT Zen</a>** руу нэвтэрч, тооцоогоо нэмнэ үү.
   дэлгэрэнгүй мэдээлэл болон API түлхүүрээ хуулна уу.
2. Та TUI-д `/connect` командыг ажиллуулж, MongolGPT Zen-г сонгоод API товчлуураа оруулна уу.
3. Бидний санал болгож буй загваруудын жагсаалтыг харахын тулд TUI дээр `/models` гэж ажиллуул.

Та хүсэлт бүрд төлбөр авах бөгөөд та дансандаа кредит нэмэх боломжтой.

### Автоматаар дахин ачаалах

Хэрэв таны үлдэгдэл 5 доллараас доош орвол Zen автоматаар 20 долларыг дахин ачаална.

Та автоматаар дахин ачаалах хэмжээг өөрчилж болно. Та мөн автоматаар дахин ачаалахыг бүхэлд нь идэвхгүй болгож болно.

### Сарын хязгаар

танай багийн гишүүн.

## Багуудад зориулав

Зэн ч гэсэн багуудад маш сайн ажилладаг. Та багын найзуудаа урих, үүрэг гүйцэтгэх, удирдах боломжтой
танай багийн ашигладаг загварууд болон бусад.

:::note
Ажлын талбарууд нь бета хувилбарын нэг хэсэг болгон багуудад одоогоор үнэ төлбөргүй байдаг.
:::

Өөрийн ажлын талбарыг удирдах нь бета хувилбарын нэг хэсэг болох багууд одоогоор үнэ төлбөргүй байна. Бид байх болно
үнийн талаар дэлгэрэнгүй мэдээллийг удахгүй хуваалцах болно.

---

### Дүрүүд

Та ажлын байрандаа багийн найзуудаа урьж, үүрэг даалгавар өгч болно.

- **Админ**: Загвар, гишүүд, API түлхүүрүүд болон тооцоог удирдах
- **Гишүүн**: Зөвхөн өөрийн API түлхүүрүүдийг удирдах

Админууд зардлаа хяналтандаа байлгахын тулд гишүүн бүрт сар бүр зардлын хязгаар тогтоож болно.

---

### Загварын хандалт

Админууд ажлын талбарт зориулсан тодорхой загварыг идэвхжүүлж эсвэл идэвхгүй болгож болно. Тахир дутуу загварт хийсэн хүсэлт нь алдаа гаргах болно.

Энэ нь тухайн загварын хэрэглээг идэвхгүй болгохыг хүссэн тохиолдолд хэрэг болно
өгөгдөл цуглуулдаг.
