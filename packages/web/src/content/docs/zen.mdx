## Одоогийн төлөв

**MongolGPT Zen одоогоор дэмжигдэхгүй.** Zen-д шаардлагатай MongolGPT-ийн өөрийн production backend, account, authentication, billing болон model gateway нийтэд ашиглахад бэлэн болоогүй.

:::caution
Zen нэртэй танихгүй домэйнд account үүсгэх, нэвтрэх мэдээлэл, API key эсвэл төлбөрийн мэдээлэл бүү оруул. MongolGPT одоогоор Zen-ийн нийтийн URL, subscription, үнэ, quota эсвэл support service санал болгохгүй.
:::

CLI, TUI, local web болон desktop ашиглахад Zen хэрэггүй. Одоогийн дэмжигдсэн зам нь [provider-ийн зааврын](../providers) дагуу OpenAI, Anthropic, Google, OpenRouter, Ollama зэрэг provider-ийг өөрийн API key, OAuth эрх эсвэл локал endpoint-оор холбох явдал юм.

```txt
/connect
/models
```

## Ирээдүйн production тохиргоо

Төслийн өөрийн backend бүрэн бэлэн болж, release notes-д Zen дэмжлэгийг албан ёсоор идэвхжүүлсэн үед л repository-ийн `.env.example`-ийг ашиглан өөрийн эзэмшлийн URL-уудыг тохируулна. Одоогоор:

```dotenv
MONGOLGPT_ENABLE_HOSTED_SERVICES=false
```

`.env.example` доторх `https://your-domain.example` нь зөвхөн placeholder бөгөөд ажилладаг service биш. Backend, authentication, authorization, billing, retention, abuse protection, monitoring болон support owner баталгаажаагүй үед `MONGOLGPT_ENABLE_HOSTED_SERVICES`-ийг `true` болгож болохгүй.

MongolGPT Go болон MongolGPT Black мөн ижил hosted backend-ээс хамаарах тул одоогоор дэмжигдэхгүй. Одоо ашиглах боломжуудыг [нээлтийн дэмжлэгийн хүрээнээс](../#нээлтийн-дэмжлэгийн-хүрээ) харна уу.
