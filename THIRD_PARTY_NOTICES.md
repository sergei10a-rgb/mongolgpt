# Гуравдагч талын мэдэгдэл

MongolGPT нь MIT лицензтэй OpenCode төслийн эх кодоос салбарлан хөгжсөн бие даасан бүтээгдэхүүн юм.

- Эх төсөл: [anomalyco/opencode](https://github.com/anomalyco/opencode)
- Эх төслийн лиценз: MIT
- Эх төслийн зохиогчийн мэдэгдэл: `Copyright (c) 2025 opencode`

MongolGPT-ийн нэр, лого, Монгол хэрэглэгчийн туршлага, нэмэлт боломжууд болон түгээлт нь MongolGPT төслийн өөрийн бүтээгдэхүүний ажил юм. Энэ repository нь GitHub дээр fork холбоосгүй бие даасан repository боловч ашигласан нээлттэй эхийн лиценз, зохиогчийн мэдэгдлийг хүндэтгэн хадгална.

Dependency бүр өөрийн лицензтэй байж болно. Тэдгээрийн лицензийн файлууд тухайн package эсвэл vendor directory дотор хадгалагдана.
